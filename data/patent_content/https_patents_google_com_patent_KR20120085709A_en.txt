Abstract
입력 검색 쿼리와 연관된 연관 검색 쿼리를 처리할 때 이미 표시된 광고들을 식별하고, 입력 검색 쿼리에 대한 검색 결과 페이지에 표시하기 위해, 연관 검색 쿼리들로부터 식별된 광고들을 제공하는 방법, 시스템 및 장치가 제공된다.Provided are a method, system, and apparatus for providing advertisements identified from associated search queries to identify advertisements that have already been displayed when processing an associated search query associated with an input search query and to display on a search results page for the input search query. do.

본 발명은 온라인 광고와 관련된 것이다. The present invention relates to online advertising.
인터넷은 특정 주제들, 도서 정보 또는 뉴스 기사에 대한 비디오 또는 오디오 파일, 웹페이지와 같은 넓고 다양한 자원들로의 접근을 제공한다. 검색 시스템은 사용자로부터 입력된 하나 이상의 검색 용어 또는 구문을 포함하는 검색 쿼리에 따라 자원들을 식별할 수 있다. 상기 검색 시스템은 상기 쿼리에 대한 그들의 연관도 및 상기 자원들의 퀄리티(quality)의 정도에 기초하여 자원들을 정렬하고, 간원들을 식별하는 검색 결과들(예를 들어, 자원들로의 링크(link))을 제공한다.The Internet provides access to a wide variety of resources, such as video or audio files and web pages on specific topics, book information or news articles. The search system may identify resources according to a search query that includes one or more search terms or phrases input from a user. The search system sorts resources based on their relevance to the query and the degree of quality of the resources, and search results (eg, links to resources) that identify appeals. To provide.
일부 검색 시스템은 또한, 사용자가 원하는 검색 결과를 찾도록 보조하기 위해 사용자에 의해 입력된 검색 쿼리와 연관된 검색 쿼리들을 제공한다. 상기 연관검색 쿼리들은 예를 들어, 검색 쿼리에 대해 알려진 정제(refinements)(예를 들어, 다른 이용자들로부터 입력된 그 쿼리들의 정제 및 과거 검색 쿼리들로부터 알려진)에 기초할 수 있다. 이 연관 검색 쿼리들은 사용자에 의해 입력된 검색 쿼리에 대응하는 검색 결과들과 함께 검색 결과 페이지에 표시될 수 있다.Some search systems also provide search queries associated with the search query entered by the user to assist the user in finding the desired search results. The associative search queries may be based, for example, on known refinements to the search query (eg, known from past search queries and refinements of those queries entered from other users). These associated search queries may be displayed in a search results page along with the search results corresponding to the search query entered by the user.
검색 결과 및/또는 연관된 검색 결과들을 제공하고, 추가적으로, 검색 시스템은 검색 쿼리와 관련된 광고들을 제공받을 수도 있다. 예를 들어, 광고 시스템은 검색 쿼리에 대한 광고들 식별하고, 상기 검색 시스템에 광고를 제공할 수 있다. 이 광고들은 검색 결과들 및 연관 검색 쿼리들과 함께 검색 결과 페이지에 표시될 수 있다.In addition to providing search results and / or associated search results, the search system may also be provided with advertisements related to the search query. For example, an advertising system may identify advertisements for a search query and provide advertisements to the search system. These advertisements may be displayed on a search results page along with the search results and associated search queries.

본 명세서에 기재된 주제의 특정 실시예는 하나 이상의 하기와 같은 장점들을 실현하기 위해 구현될 수 있다. 연관 검색 쿼리들로부터 식별된 광고들은 하나 이상의 연관 검색 쿼리들을 우선 선택해야하는 사용자없이 사용자에게 표시될 수 있다. 연관 쿼리들는 사용자 관심에 더 타겟팅된(targeted) 정제된 쿼리들이므로, 연관 쿼리들에 대응하는 광고들은 사용자 관심에 더 타겟팅된 것과 유사하다. 따라서, 사용자에 의해 입력된 검색 쿼리에 대응하여 사용자에게 표시되는 관련있는 광고들의 개수는 증가될 수 있다. 상기 연관 광고들은 선택적으로 연관 쿼리들과 함께 표시될 수 있으며, 연관 쿼리들은 사용자의 쿼리보다 더 타겟팅된 것으로 보일 쿼리들에 대응하는 광고들을 지시한다.Particular embodiments of the subject matter described in this specification can be implemented to realize one or more of the following advantages. Advertisements identified from related search queries may be displayed to the user without the user having to first select one or more related search queries. Since the association queries are refined queries that are more targeted to user interest, the advertisements corresponding to the association queries are similar to those more targeted to user interest. Thus, the number of relevant advertisements displayed to the user in response to the search query entered by the user can be increased. The associated advertisements can optionally be displayed along with associated queries, wherein the associated queries indicate advertisements corresponding to queries that appear to be more targeted than the user's query.
이와 같은 광고들의 표시 및 추가는 일부 포괄적인 키워드들이 표시하기 위한 높은 퀄리티 광고들의 충분한 개수의 식별을 야기하지 않는 문제를 해결한다. 추가적으로 페이지의 하단, 연관 쿼리들의 아래에 연관 광고들을 배치함으로써, 연관 광고들은 사용자에 입력된 쿼리에 대응하는 광고들과 시각적으로 구분된다. 이와 같은 배치는 또한, 사용자에 의해 입력된 검색 쿼리에 대응하여 표시되는 광고들 또는 검색 결과들에 사용자들이 만족하지 않는 경우, 사용자들이 페이지의 하단을 응시하는 경향이 있다는 관찰된 행동양식을 이용한다.The display and addition of such advertisements solves the problem that some generic keywords do not cause the identification of a sufficient number of high quality advertisements for display. Additionally, by placing relevant advertisements at the bottom of the page, below the associated queries, the associated advertisements are visually distinguished from advertisements corresponding to the query entered in the user. This arrangement also makes use of the observed behavior that users tend to stare at the bottom of the page if they are not satisfied with the ads or search results displayed in response to the search query entered by the user.

본 명세서는 입력 검색 쿼리와 연관된 검색 쿼리들에 대응하는 광고들을 식별하는 단계, 및 상기 입력 검색 쿼리에 대한 검색 결과 페이지를 표시하기 위해, 연관 검색 쿼리들로부터 광고들을 제공하는 단계에 관한 것이다. 입력 검색 쿼리("제1검색 쿼리)는 현재 사용자 검색 세션(session) 동안 입력된 검색 쿼리이다. 연관 검색 쿼리는 상기 입력 검색 쿼리에 유사하거나 연관된 것으로 결정되거나 분류된 검색 쿼리이다. 연관 검색 쿼리는 예를 들어, 원하는 검색 결과를 얻기 위한 유사한 검색 쿼리들 또는 상기 입력 검색 쿼리에 대해 사용자에 의해 실시된 과거의 정제로부터 결정될 수 있다. This disclosure relates to identifying advertisements corresponding to search queries associated with an input search query, and providing advertisements from associated search queries to display a search results page for the input search query. An input search query ("first search query") is a search query entered during the current user search session. An associated search query is a search query that has been determined or categorized as similar or related to the input search query. For example, it may be determined from similar search queries to obtain a desired search result or from past refinements performed by the user for the input search query.
광고들은 상기 입력 검색 쿼리 및 상기 연관 검색 쿼리들로부터 식별된다. 일부 구현들에서, 입력 검색 쿼리로부터 식별된 광고들은 검색 결과 페이지의 제1영역(예를 들어, 페이지의 우측 또는 페이지의 상단 섹션)에 표시되고, 연관 검색 쿼리들로부터 식별된 광고들("연관된 광고들"로 언급되는)은 상기 페이지의 제2영역(예를 들어, 페이지의 오른쪽 하단 및/또는 다수의 연관 검색 쿼리들의 리스트 주변)에 표시된다.

표시되는 광고들의 정렬순위은 광고에 대한 최대 클릭당 비용에 광고의 퀄리티 스코어(quality score)의 곱으로 결정될 수 있다. 광고에 대한 퀄리티 스코어는 하나 이상의 실행 매트릭스(예를 들어, 광고의 클릭률)에 비례한다. 추가적으로, 퀄리티 스코어는 입력 쿼리와 연관된 특정 키워드에 따라 변경될 수 있다. 예를 들어, 광고에 대해 높은 클릭률을 야기하는 키워드는 그 키워드에 대한 광고의 퀄리티 스코어를 증가시킬 수 있고, 반면 광고에 대해 낮은 클릭률을 야기하는 키워드는 그 키워드에 대한 광고의 퀄리티 스코어를 감소시킬 수 있다. The sorting order of the displayed advertisements may be determined by multiplying the maximum cost per click for the advertisement by the quality score of the advertisement. The quality score for the advertisement is proportional to one or more performance metrics (eg, the clickthrough rate of the advertisement). In addition, the quality score may change depending on the particular keyword associated with the input query. For example, a keyword that causes a high clickthrough rate for an ad may increase the quality score of the ad for that keyword, while a keyword that causes a low clickthrough rate for the ad may decrease the quality score of the ad for that keyword.
