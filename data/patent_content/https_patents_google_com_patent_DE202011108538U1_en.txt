Hintergrundbackground
Ein Web-Browser ist eine auf einer Rechenvorrichtung ausgeführte Softwareanwendung, die es einem Benutzer ermöglicht, Inhalt, der auf einem Netz in der Art einer Webseite auf dem World Wide Web verfügbar ist, zu betrachten oder herunterzuladen. Der Inhalt kann Text, Dateien, Bilder, Audioinformationen, Videoinformationen und persönliche Kommunikationen einschließen. Ein Browser kann es einem Benutzer auch ermöglichen, in Inhalt einzutreten, ihn hochzuladen oder ihn auszuführen. Browser können eine Anzahl von Protokollen und Standards verwenden, um einen Inhaltsfluss zu erhalten oder zu verwalten. Die meisten Browser verwenden in erster Linie das Hypertext Transfer Protocol (HTTP), um Inhalt und Webseiten abzurufen.A web browser is a software application running on a computing device that enables a user to view or download content that is available on a network, such as a web page on the World Wide Web. The content may include text, files, images, audio information, video information, and personal communications. A browser may also allow a user to enter, upload or execute content. Browsers can use a number of protocols and standards to maintain or manage a content flow. Most browsers use Hypertext Transfer Protocol (HTTP) primarily to retrieve content and web pages.
Web-Browser können verwendet werden, um Nachrichten und Sportinformationen, Wetterinformationen, Online-Foren (beispielsweise Foren sozialer Netzwerke und Geschäftsnetzwerke, Web-Blogs, Bulletin-Boards und Online-Diskussionsforen), Online-Einkaufsseiten und spezialisierte Websuchseiten und Websuchseiten für allgemeine Zwecke anzuzeigen. Web-Browser wurden verwendet, um Web-basierte Anwendungen, wie Steuererklärungssoftware, Bürosoftware (beispielsweise Software für das Präparieren von Dokumenten, Präsentationen, Spreadsheets und Datenbanken) und Spielanwendungen bereitzustellen.Web browsers can be used to display news and sports information, weather information, online forums (for example, social networking forums and business networks, web blogs, bulletin boards, and online discussion forums), online shopping pages, and specialized web search and general purpose web search pages , Web browsers have been used to provide web based applications such as tax reporting software, office software (for example, document preparation software, presentations, spreadsheets and databases) and game applications.
Web-Browser-Clients, welche ein Browsen mit Funktionsflächen unterstützen, werden immer beliebter. Mit Funktionsflächen versehene Web-Browser ermöglichen es einem Benutzer, mehrere Browserfenster zu öffnen, die jeweils eine entsprechende Funktionsfläche für das Auswählen des Fensters aufweisen. Die Funktionsflächen können in einem Funktionsflächenbereich des Web-Browser-Clients angeordnet sein. Typischerweise belegen die Funktionsflächen einen erheblich kleineren Bereich des Browsers als die Browserfenster. Der kleine Raum, der von den Funktionsflächen belegt wird, ermöglicht es, dass eine große Anzahl mit Funktionsflächen versehener Fenster auf einmal geöffnet wird, ohne den für die Funktionsflächen reservierten Platz zu verbrauchen oder den Platz für das Anzeigen eines aktiven Browserfensters erheblich zu verringern.Web browser clients that support browsing with functional surfaces are becoming increasingly popular. Functional Web browsers allow a user to open multiple browser windows, each with a corresponding functional area for selecting the window. The functional areas may be arranged in a functional area of the web browser client. Typically, the functional areas occupy a significantly smaller area of the browser than the browser windows. The small space occupied by the functional areas allows a large number of functionalized windows to be opened at once, without consuming the space reserved for functional areas or significantly reducing the space for displaying an active browser window.
Benutzer können durch Auswählen von Browser-Funktionsflächen wirksam zwischen Anwendungen schalten. Wenn die Anzahl der Funktionsflächen zunimmt, verringert der Aufwand, der benötigt wird, um die geeignete Browser-Funktionsfläche zu lokalisieren und auszuwählen, die Effizienz des Benutzers.Users can effectively switch between applications by selecting browser functional areas. As the number of functional areas increases, the effort required to locate and select the appropriate browser functional area reduces the efficiency of the user.

Diese Möglichkeit kann auch das Organisieren und Anzeigen von Funktionsflächensätzen bieten.Embodiments are provided for managing the functional windows for multiple windows on a client based on a user task. This option can also be used to organize and display feature sets.
