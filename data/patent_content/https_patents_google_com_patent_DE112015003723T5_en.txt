BESCHREIBUNG DES STANDES DER TECHNIK DESCRIPTION OF THE PRIOR ART
Digitale Vorrichtungen präsentieren lesbaren digitalen Inhalt wie Romane, Zeitungsartikel und Kurzerzählungen sowie digitalen Inhalt wie Videos und Bilder. Gegenwärtige Schnittstellen zum Navigieren von E-Books stellen eine lineare Navigation und eine grobe Navigation bereit. Diese Schnittstellen sind dazu geeignet, um Romane zu lesen, sind aber weniger dafür geeignet, Fachbücher wie Lehrbücher zu lesen oder um Sammlungen von Inhalt wie Kochbücher oder Magazine zu lesen. Beispielsweise greift ein Benutzer, der einen Abschnitt in einem Lehrbuch liest, auf ein Glossar zurück, er kann aber nicht bequem zu dem Abschnitt zurückkehren, den der Benutzer gelesen hatte. Digital devices present readable digital content such as novels, newspaper articles and short stories as well as digital content such as videos and images. Current interfaces for navigating e-books provide linear navigation and navigation. These interfaces are suitable for reading novels, but are less likely to read textbooks such as textbooks or to read collections of content such as cookbooks or magazines. For example, a user reading a section in a textbook accesses a glossary, but can not easily return to the section that the user had read.

Ein E-Book ist ein elektronisches Buch, das Inhalt und Metadaten umfasst. Der E-Book-Inhalt umfasst eine elektronische Textsammlung (und möglicherweise Nicht-Textinhalt wie Bilder, Animationen, Sound und/oder Video). Beispielsweise umfasst der E-Book-Inhalt den Inhalt eines Buches, wie er auf den Seiten eines traditionellen papierbasierten Buches gedruckt werden würde. Der gesamte Text in der Sammlung kann von einer einzelnen Quelle sein oder die Sammlung kann Text von unterschiedlichen Quellen enthalten. Der Text kann von einem traditionellen papierbasierten Format (z. B. ein physisches Buch oder eine Zeitung) oder von einem digitalen Format herstammen (z. B. elektronische Dateien in einem Format wie Portable Document Format (PDF), Electronic Publication-(EPUB)-Format oder Fließtextformat). An e-book is an electronic book that includes content and metadata. The e-book content includes an electronic text collection (and possibly non-text content such as images, animations, sound, and / or video). For example, the e-book content includes the contents of a book as it would be printed on the pages of a traditional paper-based book. All the text in the collection may be from a single source or the collection may contain text from different sources. The text may come from a traditional paper-based format (eg a physical book or a newspaper) or from a digital format (eg electronic files in a format such as Portable Document Format (PDF), Electronic Publication (EPUB)). Format or continuous text format).
Ein Abschnitt eines E-Books ist ein grundlegendes Gruppieren des Inhaltes innerhalb eines E-Books. Ein Abschnitt (z. B. ein Wort, ein Satz, Nicht-Text-Inhalt) eines E-Books ist mit einer Stelle verbunden, welche die sequenzielle Sortierung dieses Abschnitts relativ zu anderen Abschnitten des E-Books anzeigt. Abschnitte eines E-Books sind in Segmente angesammelt, die das E-Book in Gruppierungen von in Beziehung stehenden Abschnitten aufteilen. Ein Segment kann beispielsweise ein Kapitel, ein Unterabschnitt eines Kapitels, ein Index, eine Karte, ein Rezept, ein Literaturverzeichnis, ein Artikel, ein Glossar oder eine Kombination davon sein. Beispielsweise kann ein E-Book ein Segment für jedes Kapitel des E-Books und ein Segment für den Index umfassen. A section of an ebook is a basic grouping of content within an ebook. A portion (eg, a word, a sentence, non-text content) of an ebook is associated with a location that indicates the sequential sorting of that portion relative to other portions of the ebook. Portions of an ebook are grouped into segments that divide the ebook into groupings of related sections. For example, a segment may be a chapter, a subsection of a chapter, an index, a map, a recipe, a bibliography, an article, a glossary, or a combination thereof. For example, an e-book may include one segment for each chapter of the ebook and one segment for the index.
Die E-Book-Metadaten umfassen bibliografische Informationen über das E-Book, wie beispielsweise der E-Book-Bezeichner, Titel, Offenlegungstag, Ausgabe/Versionsnummer, Seitenanzahl, Anzahl an Kapiteln, eine Segmentauflistung, Seitenminiaturansichten für E-Book-Seiten oder eine Kombination davon. The e-book metadata includes bibliographic information about the e-book, such as the e-book identifier, title, disclosure date, issue / version number, page count, number of chapters, segment listing, page thumbnails for e-book pages, or one Combination of it.

A navigation interface is a GUI that allows a user to navigate to different e-book pages in a fast and / or non-linear manner. A navigation command can cause the navigation module 150 switches between the navigation interface and the read interface. An exemplary navigation interface will be described in relation to FIG 3A to 3C described below.
Das Navigationsmodul 150 erzeugt einen intelligenten Scrubber als Teil der Navigationsschnittstelle. Der intelligente Scrubber zeigt einen Fortschritt durch das E-Book grafisch an. Eine gegebene Position auf dem intelligenten Scrubber entspricht einer Stelle eines Abschnitts des E-Books. Beispielsweise ist der intelligente Scrubber ein horizontaler Balken mit einem linken Ende, das die erste E-Book-Seite darstellt, und einem rechten Ende, das die letzte E-Book-Seite darstellt. Als ein weiteres Beispiel ist der intelligente Scrubber eine Linie mit Positionen auf der Linie, die unterschiedliche Stellen im E-Book darstellt. Als Reaktion auf Navigationsbefehle, die an den intelligenten Scrubber gerichtet sind, zeigt das Navigationsmodul 150 einen anderen Abschnitt des Buches an, ohne zwangsläufig linear durch die dazwischen befindlichen Abschnitte des E-Books zu navigieren. The navigation module 150 creates a smart scrubber as part of the navigation interface. The smart scrubber graphically displays progress through the e-book. A given position on the smart scrubber corresponds to a location of a section of the ebook. For example, the smart scrubber is a horizontal bar with a left end representing the first e-book page and a right end representing the last e-book page. As another example, the smart scrubber is a line with positions on the line representing different places in the e-book. In response to navigation commands directed to the intelligent scrubber, the navigation module shows 150 another section of the book, without necessarily linearly navigating through the intervening sections of the e-book.
Das Navigationsmodul 150 kann kontextabhängige Symbole auf dem intelligenten Scrubber anzeigen. Ein kontextabhängiges Symbol stellt kontextabhängige Informationen dar, die mit einer Stelle im E-Book verbunden sind. Kontextabhängige Informationen ergänzen den Inhalt eines Abschnitts des E-Books und sind mit der Stelle dieses Abschnitts verbunden. Ein kontextabhängiges Symbol, das kontextabhängige Informationen darstellt, ist mit der Stelle dieser kontextabhängigen Informationen verbunden. Die Position eines kontextabhängigen Symbols auf dem Scrubber entspricht der mit dem kontextabhängigen Symbol verbundenen Stelle. Ein kontextabhängiges Symbol kann das Vorhandensein von kontextabhängigen Informationen wie eine Anmerkung (z. B. ein Lesezeichen, ein Highlight, ein Kommentar), den Anfang eines Segments (z. B. die erste E-Book-Seite eines Kapitels) oder einen Benutzerfortschritt (z. B. eine gegenwärtige Lesestelle, ein am weitesten gelesene Stelle, eine Stelle am Anfang des gegenwärtigen Lesevorgangs des Benutzers) anzeigen. Kontextabhängige Symbole können mit einem Benutzer der Benutzervorrichtung 100, einem anderen Benutzer einer anderen Lesevorrichtung 100 oder einer Ansammlung von Benutzern verbunden sein. Beispielsweise zeigt ein kontextabhängiges Symbol die Stelle von beliebten Highlights oder Lesezeichen unter Benutzern des E-Reader-Moduls 140 an. Als ein weiteres Beispiel zeigt ein kontextabhängiges Symbol eine gegenwärtige Lesestelle einer Verbindung mit einem sozialen Netzwerk des Benutzers an. The navigation module 150 can display contextual icons on the smart scrubber. A contextual icon represents contextual information associated with a location in the ebook. Contextual information complements the content of a section of the ebook and is linked to the section of this section. A contextual icon representing contextual information is associated with the location of this contextual information. The position of a contextual icon on the scrubber corresponds to the location associated with the contextual icon. A contextual icon may indicate the presence of contextual information such as a comment (eg, a bookmark, a highlight, a comment), the beginning of a segment (eg, the first e-book page of a chapter), or a user's progress ( for example, a current reading location, a most read place, a location at the beginning of the user's current reading operation). Context-dependent icons may be shared with a user of the user device 100 , another user of another reading device 100 or a collection of users. For example, a contextual icon shows the location of popular highlights or bookmarks among users of the e-reader module 140 at. As another example, a contextual icon indicates a current read location of a connection to a social network of the user.
Bei einer Ausführungsform interpretiert das Navigationsmodul 150 Navigationsbefehle auf dem intelligenten Scrubber basierend auf kontextabhängigen Symbolen nahe dem Navigationsbefehl. Beispielsweise detektiert das Navigationsmodul 150 einen Navigationsbefehl, der an eine Position auf dem Scrubber gerichtet ist, die sich nahe an einem angezeigten kontextabhängigen Symbol, das den Anfang eines Kapitels bezeichnet, befindet. Als Reaktion auf den Navigationsbefehl zeigt das Navigationsmodul 150 den Beginn des Kapitels an anstatt die tatsächliche E-Book-Seite, die der Position des Navigationsbefehls entspricht. In one embodiment, the navigation module interprets 150 Navigation commands on the intelligent scrubber based on contextual symbols near the navigation command. For example, the navigation module detects 150 a navigation command directed to a position on the scrubber that is close to a displayed contextual symbol that marks the beginning of a chapter. In response to the navigation command, the navigation module shows 150 the beginning of the chapter instead of the actual e-book page that corresponds to the position of the navigation command.
Bei einer Ausführungsform zeigt die Navigationsschnittstelle E-Book-Seiten als Miniaturansichten an. Nachdem ein Navigationsbefehl, der eine Navigationsschnittstelle anfordert, empfangen wurde, identifiziert das Navigationsmodul 150 E-Book-Seiten für die Miniaturansichterzeugung und erzeugt Seitenminiaturansichten für die identifizierten E-Book-Seiten. Eine Seitenminiaturansicht ist ein wählbares Bild einer E-Book-Seite, die in der Größe verkleinert wurde und die bei Auswahl bewirkt, dass das E-Reader-Modul 140 die E-Book-Seite anzeigt, die mit der Seitenminiaturansicht in der Leseschnittstelle verbunden ist. Bei einer Ausführungsform erzeugt das Navigationsmodul 150 eine Seitenwiederherstellungsminiaturansicht der in der Leseschnittstelle angezeigten letzten Seite. Die Seitenwiederherstellungsminiaturansicht ist ein kontextabhängiges Symbol, das auf oder neben dem intelligenten Scrubber an einer Position, die der Stelle der letzten gelesenen Seite entspricht, angezeigt wird. Indem der Benutzer die Seitenwiederherstellungsminiaturansicht auswählt, kann ein Benutzer zur letzten gelesenen Seite in der Leseschnittstelle zurückkehren. In one embodiment, the navigation interface displays e-book pages as thumbnails. After a navigation command requesting a navigation interface has been received, the navigation module identifies 150 Thumbnail creation e-book pages and generate page thumbnails for the identified e-book pages. A page thumbnail is a selectable image of an ebook page that has been resized and that when selected causes the eReader module 140 displays the ebook page associated with the page thumbnail in the read interface. In one embodiment, the navigation module generates 150 a page restore thumbnail view of the last page displayed in the read interface. The page-recovery thumbnail is a context-sensitive icon that appears on or near the smart scrubber at a location that corresponds to the location of the last read page. By selecting the page restore thumbnail view, a user can return to the last read page in the read interface.

Eine Auf-Zuziehen-Geste oder eine Strecken-Geste wird empfangen, wenn zwei Stellen auf der Touchscreenschnittstelle berührt werden und sich dann ein Abstand zwischen den zwei Stellen ändert. Ein Bezugsabstandswert ist ein Wert, der mit dem ursprünglichen Abstand zwischen zwei Stellen verbunden ist, die ursprünglich auf einer Touchscreenschnittstelle berührt wurden. Ein aktiver Abstandswert ist ein Wert, der mit dem gegenwärtigen Abstand zwischen zwei berührten Stellen verbunden ist. Eine Auf-Zuziehen-Geste bezeichnet typischerweise eine Geste, bei der der aktive Abstandswert relativ zum Bezugsabstandswert abnimmt. Eine Strecken-Geste bezeichnet eine Geste, bei der aktive Abstandswert relativ zum Bezugsabstandswert zunimmt. An open-to-close gesture or a stretch gesture is received when two locations on the touch screen interface are touched and then a distance between the two locations changes. A reference distance value is a value associated with the original distance between two locations that were originally touched on a touch screen interface. An active distance value is a value associated with the current distance between two touched locations. An open-to-close gesture typically refers to a gesture in which the active distance value decreases relative to the reference distance value. A stretch gesture refers to a gesture in which the active distance value increases relative to the reference distance value.
