Description of the invention: [Technical field to which the invention pertains] This specification relates to a language input method. [Prior Art] The language input method allows the user to provide a language input using an input method editor (IME)*. The IME is allowed to use one of the input methods in one of the language 'input method applications'. An input method allows a user to provide a target language input (e.g., a non-Latin language) using a keyboard (e.g., a QWERTY keyboard) in an source language. The IME maps the ideogram of the target language to a key on the source language keyboard (eg, according to the mapping, pressing a specific key or key combination in an input language to generate a corresponding ideogram in the target language.) It is known that the IME only replaces the mapped character; it does not translate the input (eg, interpreting one of the input words in a source language and providing a translated word with a substantially identical meaning in a target language). When a user is willing to provide input from a source language and translate the input into a target language, the user typically uses a separate dictionary or translation application. The user will provide input to the dictionary or transfer application in the source language. The user then copies the translation input (e.g., translated into the target language) from the dictionary or translation application and pastes the input of the translation into the desired location. In addition, if the user wants to transfer the separated part of the translated word (the two columns are different lines of the chat conversation), the user usually provides each of the texts to the dictionary or translation application. The user will then copy each part of the translated text from the dictionary or translation application and paste it in the desired location (eg, in the chat interface). 138573.doc 200947237 SUMMARY OF THE INVENTION [0007] Provided are systems, methods, and apparatus for computer program products, particularly for translating input.
