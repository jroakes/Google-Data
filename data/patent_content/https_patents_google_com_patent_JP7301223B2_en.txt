The solutions described herein also reduce privacy risks (eg, data leakage or unauthorized access to user data) associated with storing raw user data. Typically, providers may store user data in its raw form, but such raw data storage is susceptible to unauthorized access. For example, if a malicious party gains access to a provider's system, raw user data stored in such system may be easily accessible. The solution described herein reduces this risk by inserting privacy-enhanced representations of user data, such as hashed representations of user data, into probabilistic data structures that further mask user data. More specifically, the combination of using hashed representations of user data in combination with probabilistic data structures not only masks the data itself (e.g., using hashed representations rather than actual user data), It also masks the number of users with a particular set of user data, thereby providing enhanced data privacy protection.
