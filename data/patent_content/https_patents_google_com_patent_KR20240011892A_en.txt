Abstract
모호한 쿼리에 대한 관련 응답을 전달하는 방법(200)은 사용자(10)와 연관된 사용자 디바이스(110)로부터, 디지털 어시스턴트 서비스(160)로부터의 정보 또는 상기 디지털 어시스턴트 서비스. 에 의한 액션 수행을 요청하는 쿼리(120)를 수신하는 단계를 포함한다. 상기 쿼리의 사용자 의도가 해결되지 않은 경우, 상기 방법은 검색 엔진(164)으로부터, 상기 쿼리(120)에 대한 현재 트렌딩 검색 결과를 검색하는 단계; 및 상기 검색 결과에 기초하여, 상기 쿼리에 대한 사용자 의도를 해결하는 단계를 포함한다. 방법은 또한 상기 해결된 사용자 의도에 기초하여 상기 쿼리에 대한 응답(122)을 생성하는 단계를 포함한다. 상기 응답은 상기 해결된 사용자 의도와 연관된 특정 의도 버티컬로부터 획득된 정보 또는 상기 디지털 어시스턴트 서비스에 의해 액션의 이행을 포함한다.A method 200 of delivering a relevant response to an ambiguous query may be a method of delivering a relevant response to an ambiguous query from a user device 110 associated with a user 10, information from a digital assistant service 160, or the digital assistant service. It includes receiving a query 120 requesting to perform an action. If the user intent of the query is not resolved, the method includes retrieving, from a search engine (164), currently trending search results for the query (120); and resolving user intent for the query based on the search results. The method also includes generating a response 122 to the query based on the resolved user intent. The response includes information obtained from a specific intent vertical associated with the resolved user intent or execution of an action by the digital assistant service.

본 개시는 어시스턴트에 대한 사용자 의도를 이해하기 위한 검색 결과 기반 트리거링에 관한 것이다.This disclosure relates to search result-based triggering to understand user intent toward an assistant.
사용자는 토픽/엔티티에 관한 정보 및 사실을 얻거나 사용자가 특정 작업을 수행하는 것에 도움을 주기 위해 컴퓨팅 디바이스에서 실행되는 디지털 어시스턴트에 쿼리할 수 있다. 디지털 어시스턴트는 사용자가 관심을 갖는 특정 정보를 찾도록 디지털 어시스턴트를 안내하기 위해 사용자가 충분한 정보를 제공할 수 있을 것을 요구할 수 있다. 쿼리가 충분히 맞춤화되지 않은 경우 또는 사용자가 쿼리와 관련된 엔터티를 단순히 지정하는 것 이상의 추가 정보 방식으로 많이 제공하지 않는 경우, 디지털 어시스턴트는 쿼리를 더욱 좁히고 사용자 의도를 얻기 위해 명확하게 하는 질문을 사용자에게 프롬프트할 수 있다. 추가적으로, 디지털 어시스턴트는 동적으로 업데이트되어야 하는 기본 응답을 제공함으로써 사용자 의도가 부족한 모호한 쿼리에 대해 미리 결정된 쿼리 해석에 의존할 수 있다.A user may query a digital assistant running on the computing device to obtain information and facts about a topic/entity or to assist the user in performing a specific task. The digital assistant may require the user to provide sufficient information to guide the digital assistant to find specific information of interest. If the query is not tailored enough, or if the user does not provide much in the way of additional information beyond simply specifying the entities relevant to the query, the digital assistant will prompt the user with clarifying questions to further narrow the query and obtain user intent. can do. Additionally, digital assistants can rely on predetermined query interpretation for ambiguous queries that lack user intent by providing default responses that must be dynamically updated.

A computer-implemented method when executed on data processing hardware causes the data processing hardware to perform operations, the operations comprising receiving, from a user device associated with a user, a query requesting information from a digital assistant service; and when the user intent of the query is not resolved: retrieving, from a search engine, current trending search results for the query; Based on the search results, resolving user intent for the query; and generating a response to the query based on the resolved user intent, wherein the response includes information obtained from a specific intent vertical associated with the resolved user intent.

In such implementations, performing query interpretation on the query to determine whether the user intent is ambiguous includes processing the query to determine individual scores for each of one or more possible user intents of the query. ; and determining that the user intent is ambiguous if the individual score determined for each of the one or more possible user intents of the query does not meet a confidence threshold. In some examples, based on the search results, resolving user intent for the query includes identifying a search result in a first position in a currently trending search result for the query retrieved from the search engine, the first determining at least one of an entity or search result type associated with the search result at a position, and resolving the user intent based on at least one of an entity or search result type associated with the search result at the first position.
