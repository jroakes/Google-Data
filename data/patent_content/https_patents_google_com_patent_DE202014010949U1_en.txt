Abstract
Ein Computersystem für Sonderziele (Points of Interest – POIs) für die Anzeige auf einer digitalen Karte an einem Client-Computergerät basierend auf Kontextinformationen im Zusammenhang mit einem Benutzer, wobei das System Folgendes umfasst: ein POI-Auswahlsystem in Kommunikation mit einem Speicher zum Speichern von Anweisungen zur Ausführung des POI-Auswahlsystems auf einem Prozessor, wobei die Anweisungen Folgendes enthalten: Empfang von Standortdaten von einem Client-Computergerät, wobei die Standortdaten Anzeige einer geografischen Position des Computergeräts, Festlegung einer Benutzer-Klassifizierung basierend auf den Standortdaten, wobei die Benutzer-Klassifizierung einen Grad an Vertrautheit mit dem Standort angibt, der von den Standortdaten angegeben wird und Festlegung von Priorisierungsdaten auf Basis von Standortdaten und Benutzer-Klassifizierung, wobei die Priorisierungsdaten einen oder mehrere Sätze von zu priorisierenden POIs angeben; ein Abbildungssystem in Kommunikation mit einem Speicher, der Anweisungen speichert zur Ausführung auf einem Prozessor des Mapping-Systems, wobei die Anweisungen umfassen: Senden der Priorisierungsdaten an das Client-Computergerät, worin die Priorisierungsdaten die Anzeige auf dem Client-Computergerät des oder der angegebenen POI-Sätze ermöglicht, anschließend Empfang einer Anforderung für Mapping-Daten, worin die angeforderte Mapping-Daten dem Standort eines priorisierten POI entsprechen und Senden der angeforderten Mapping-Daten an das Client-Computergerät in Reaktion auf die nachfolgende Anforderung.A points of interest (POIs) for display on a digital map at a client computing device based on contextual information associated with a user, the system comprising: a POI selection system in communication with a memory for storing Instructions for executing the POI selection system on a processor, the instructions including: receiving location data from a client computing device, the location data indicating a geographic location of the computing device, determining a user classification based on the location data, wherein the user Classifying a degree of familiarity with the location indicated by the location data and determining prioritization data based on location data and user classification, the prioritization data indicating one or more sets of POIs to be prioritized; an imaging system in communication with a memory storing instructions for execution on a processor of the mapping system, the instructions comprising: sending the prioritization data to the client computing device, wherein the prioritization data is the indication on the client computing device of the specified POI or POIs Records, then receiving a request for mapping data, wherein the requested mapping data corresponds to the location of a prioritized POI, and sending the requested mapping data to the client computing device in response to the subsequent request.

Das Computersystem und das Verfahren, die in dieser Offenbarung beschrieben werden, schlagen dem Benutzer einer Kartierungsanwendung auf einem Computergerät Sonderziele (Points of Interest (POIs)) vor. Diese Vorschläge werden basierend auf Kontextinformationen wie beispielsweise der Standort des Benutzers, der Vertrautheit des Benutzers mit dem Standort, Uhrzeit, Informationen über die POIs und andere Benutzerprofildaten festgelegt. Informationen zu den POIs, wie beispielsweise Kategorien, Unterkategorien, Empfehlungen und Entfernung, können verwendet werden, um die Vorschläge zu ermitteln und Vorschläge in POI-Kategorien zu organisieren. Sobald dem Benutzer Vorschläge für POIs und/oder POI-Kategorien gemacht wurden, kann er durch die Vorschläge navigieren und daraufhin weitere Informationen zu den Vorschlägen anfordern. Wenn der Benutzer ein vorgeschlagenes POI ausgewählt hat, werden Informationen zu diesem bestimmten POI angefordert und das POI wird in der Kartierungsanwendung auf dem Computergerät angezeigt.The computer system and method described in this disclosure suggest Points of Interest (POIs) to the user of a mapping application on a computing device. These suggestions are determined based on contextual information such as the location of the user, the user's familiarity with the location, time, information about the POIs, and other user profile information. Information about the POIs, such as categories, subcategories, recommendations, and distance, can be used to identify the suggestions and organize suggestions into POI categories. Once the user has made suggestions for POIs and / or POI categories, they can navigate through the suggestions and then request further information about the proposals. If the user has selected a suggested POI, information about that particular POI is requested and the POI is displayed in the mapping application on the computing device.
