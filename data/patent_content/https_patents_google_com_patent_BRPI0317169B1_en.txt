Abstract
"método e dispositivo para escolha de um algoritmo de seleção de sistema que é dependente da localização". é descrito uma unidade de comunicação sem fio 200, e um correspondente método na mesma, que é disposta e construída para escolher um algoritmo de seleção de sistema que é usado para selecionar um sistema para fornecer serviço para a unidade de comunicação sem fio e compreende: meio para associar uma localização com a unidade de comunicação sem fio 223, 215; um controlador 209 para escolher um algoritmo de seleção de sistema a partir de uma pluralidade de algoritmos com base na localização, onde o algoritmo escolhido inclui parâmetros de varredura (fig.3) que variam com a localização; e um receptor 205 controlado pelo controlador de acordo com o algoritmo escolhido para pesquisar um sistema para fornecer serviço para a unidade de comunicação sem fio; e método correspondente dentro da mesma."method and device for choosing a system selection algorithm that is location dependent". A wireless communication unit 200 is described, and a corresponding method therein, which is arranged and constructed to choose a system selection algorithm that is used to select a system to provide service to the wireless communication unit and comprises: means for associating a location with the wireless communication unit 223, 215; a controller 209 for choosing a system selection algorithm from a plurality of location-based algorithms, wherein the chosen algorithm includes scan parameters (fig.3) that vary with location; and a controller-controlled receiver 205 according to the algorithm chosen to search a system for providing service to the wireless communication unit; and corresponding method within it.

BACKGROUND OF THE INVENTION
Equipamentos e sistemas de comunicação são conhecidos. Sistemas fornecidos por muitas portadoras diferentes operando de acordo com muitas tecnologias de acesso diferentes em várias frequências diferentes tornaram-se disponíveis e usados de maneira eficaz. Em muitas áreas, sistemas múltiplos são disponíveis para fornecer serviço para uma unidade de comunicação sem fio ou unidade de assinante. Os sistemas que estão disponíveis dentro de uma determinada área podem mudar com o tempo.Communication equipment and systems are known. Systems provided by many different carriers operating according to many different access technologies on several different frequencies have become available and used effectively. In many areas, multiple systems are available to provide service for a wireless communication unit or subscriber unit. The systems that are available within a given area may change over time.
Quando uma unidade de comunicação sem fio é ativada ou ligada ou entra em uma nova área, ela usa tipicamente um algoritmo de seleção de sistema para localizar ou selecionar um sistema e tenta acessar o mesmo. O algoritmo de seleção de sistema é usado para pesquisar um sistema, especificamente ajusta a unidade para várias frequências e determina se um sinal esperado ou compatível está presente, por exemplo, pode ser recebido. Achar o sinal correto, consequentemente o sistema, pode consumir muito tempo e ser uma tarefa de alto consumo de energia devido à proliferação dos sistemas diferentes.When a wireless communication unit is activated or turned on or enters a new area, it typically uses a system selection algorithm to locate or select a system and attempts to access it. The system selection algorithm is used to search for a system, specifically sets the unit to various frequencies and determines whether an expected or compatible signal is present, for example, can be received. Finding the correct signal, therefore the system, can be very time consuming and a high energy consumption task due to the proliferation of different systems.
