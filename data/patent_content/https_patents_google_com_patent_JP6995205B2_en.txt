今日、コンピュータ、スマートフォン、埋め込みデバイスなどで実行されるソフトウェアアプリケーションは、トラフィックデータを含む可能性のある段階的なナビゲーション方向を生成する。典型的には、ユーザが開始位置および宛先位置を指定し、ソフトウェアアプリケーションは、直ちに、および/または、ユーザが開始位置から宛先位置まで移動するにつれて、道順を表示する。ソフトウェアアプリケーションは、典型的には、最短の時間量で宛先位置までユーザを連れて行くルートの道順を生成する。最短の時間量は、ルートの様々な区間をトラバースするための予測時間量に基づいてもよい。 Today, software applications running on computers, smartphones, embedded devices, etc. generate step-by-step navigation directions that may contain traffic data. Typically, the user specifies a start position and a destination position, and the software application displays directions immediately and / or as the user moves from the start position to the destination position. Software applications typically generate route directions that take the user to a destination location in the shortest amount of time. The shortest amount of time may be based on the estimated amount of time to traverse different sections of the route.

More specifically, the ex post facto route is determined by calculating the amount of real time it takes for the user to traverse each road section included in the route according to the positioning data from the user's client device. The performance evaluation server also determines the amount of real time it takes for another user to traverse an alternative road section that is not included in the route. The performance evaluation server then identifies the ex post facto route as a combination of the road section on the actual route and the alternative road section, and the combination provides the route with the shortest amount of time to the destination location.

For example, the display may indicate that the navigation application identifies the fastest route with 95% time. Performance metrics may also be used to identify errors in the routing algorithm and perform root cause analysis to determine the cause of the errors. In addition, performance metrics may be used to tune the route determination model used to generate routes from start to destination. For example, a route determination model may include a combination of parameters for determining whether a particular road section is included in the route.
