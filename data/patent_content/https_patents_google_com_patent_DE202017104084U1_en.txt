Abstract
Vorrichtung, bestehend aus mindestens einem Prozessor und mindestens einem Speicher einschließlich einem Computerprogrammcode, der bei Ausführung durch den Prozessor die Vorrichtung zur Durchführung eines Verfahrens veranlasst, umfassend: Empfangen einer abfrageunabhängigen Anfrage nach kontextbezogene Informationen von einem Benutzergerät durch einen Server für Kontextinformationen, die relevant für eine aktive Ressource sind, die in einer Anwendungsumgebung auf dem Benutzerendgerät angezeigt wird, wobei die Anfrage Inhalt der aktiven Ressource spezifiziert und weiter spezifiziert, dass die auf dem Benutzergerät angezeigte aktive Ressource ein Bild beinhaltet, aber das Bild nicht in die Anfrage einbezieht; Bestimmen durch den Server für Kontextinformationen, dass der durch die Anfrage spezifizierte Inhalt nicht ausreichend ist, um kontextbezogene Informationen über die aktive Ressource bereitzustellen; als Reaktion auf das Bestimmen, dass der durch die Anfrage spezifizierte Inhalt nicht ausreichend ist, um kontextbezogene Informationen über die aktive Ressource bereitzustellen, Bereitstellen einer Anfrage an das Benutzergerät nach dem Bild in der aktiven Ressource durch den Server für Kontextinformationen; Empfangen des Bildes in der aktiven Ressource durch den Server für Kontextinformationen von dem Benutzergerät; Bestimmen eines Benutzeroberflächenelements durch den Server für Kontextinformationen, das kontextbezogene Informationen über das Bild beinhaltet, und Bereitstellen des Benutzeroberflächenelements durch den Server für Kontextinformationen für das Benutzergerät.An apparatus comprising at least one processor and at least one memory including computer program code that, when executed by the processor, causes the device to perform a method, comprising: receiving a query-independent request for contextual information from a user device by a context information server relevant to an active resource displayed in an application environment on the user terminal, the request specifying content of the active resource and further specifying that the active resource displayed on the user equipment includes an image but does not include the image in the request; Determining, by the context information server, that the content specified by the request is insufficient to provide contextual information about the active resource; in response to determining that the content specified by the request is not sufficient to provide contextual information about the active resource, providing a request to the user device for the image in the active resource by the context information server; Receiving the image in the active resource by the context information server from the user equipment; Determining by the server for context information a user interface element that includes contextual information about the image and providing the user interface element by the server for context information for the user device.

HINTERGRUND BACKGROUND
Diese Spezifikation betrifft das Bereitstellen kontextbezogener Informationen für einen Benutzer. This specification relates to providing contextual information to a user.
Ein Gerät kann einem Benutzer kontextbezogene Informationen für eine angezeigte Ressource bereitstellen, die ein Bild beinhaltet. Zum Beispiel, kann ein Gerät eine Webseite über ein bestimmtes Thema anzeigen, eine Suchabfrage vom Benutzer über das bestimmte Thema empfangen, Suchergebnisse in Reaktion auf die Suchabfrage abrufen und die Suchergebnisse dem Benutzer bereitstellen. Typische Interaktionsmodelle verlangen vom Benutzer eine Art von Benutzerabfrage an ein Benutzergerät. Zum Beispiel kann ein Benutzer ein Bild von der Webseite speichern und das Bild dann als Abfragebild hochladen. Ein Suchprozess führt dann eine Suche nach Ressourcen unter Verwendung des Abfragebildes aus, um Ressourcen zu identifizieren, die das Informationsbedürfnis des Benutzers befriedigen. A device may provide a user with contextual information for a displayed resource that includes an image. For example, a device may display a web page about a particular topic, receive a search query from the user about the particular topic, retrieve search results in response to the search query, and provide the search results to the user. Typical interaction models require a type of user query to a user device from the user. For example, a user may save an image from the web page and then upload the image as a query image. A search process then performs a search for resources using the query image to identify resources that satisfy the user's information needs.

Bestimmte Ausführungsformen des in dieser Spezifikation beschriebenen Gegenstands können so implementiert werden, dass sie einen oder mehrere der folgenden Vorteile verwirklichen. Kontextbezogene Informationen, die wahrscheinlich ein Informationsbedürfnis des Benutzer befriedigen würden, wenn er eine Ressource, die ein Bild beinhaltet, betrachtet, können durch ein Gerät für einen Benutzer bereitgestellt werden, ohne dass der Benutzer eine Abfrage an das Gerät bereitstellt. Dies resultiert in einem komfortablen Weg für den Benutzer, um kontextbezogene Informationen für eine angezeigte Ressource zu erhalten. Die Benutzer muss keine Begriffe für eine Abfrage eingeben, um kontextbezogene Informationen zu erhalten, oder ein Bild in einer angezeigten Ressource speichern und dann das gespeicherte Bild hochladen, um kontextbezogene Informationen zu erhalten. Dementsprechend erhalten Benutzer kontextbezogene Informationen in einer Weise, die für die Benutzer komfortabel ist. Außerdem, da der Benutzer keine Anfrage bereitstellen muss, kann der Benutzer in einigen Implementierungen kontextbezogene Informationen empfangen, wenn der Benutzer ansonsten nicht in der Lage wäre, eine Abfrage effektiv zu tippen, z. B. wenn der Benutzer nur eine Hand frei hat. Certain embodiments of the subject matter described in this specification may be implemented to achieve one or more of the following advantages. Contextual information that would likely satisfy an informational need of the user when he / she has a resource that includes an image, can be provided by a device to a user without the user providing a query to the device. This results in a convenient way for the user to obtain contextual information for a displayed resource. Users do not have to enter terms for a query to obtain contextual information, or save an image in a displayed resource and then upload the stored image to obtain contextual information. Accordingly, users obtain contextual information in a manner that is comfortable for the users. In addition, since the user does not need to provide a request, in some implementations the user may receive contextual information if the user would otherwise be unable to effectively type a query, e.g. B. if the user has only one hand free.
Da die kontextbezogenen Informationen ohne Abfrage bereitgestellt werden können, führt dies zu weniger Eingabefehlern und irrtümlich eingegebenen Abfragen. Dementsprechend werden in der Gesamtbetrachtung Tausende von fehlerhaften und ungenauen Abfragen vermieden, was wiederum eine effektivere Verwendung von Suchsystemressourcen gestattet. In anderen Worten werden mehrere fehlerhafte Abfrageverarbeitungszyklen vermieden, was die erforderlichen Verarbeitungsressourcen reduziert und die gesamten Systembandbreitenanforderungen reduziert (oder, alternativ, die Bedienung einer größere Anzahl von Benutzern ohne entsprechende Erhöhung von Verarbeitungsressourcen ermöglicht). Diese Verbesserung auf dem technischen Gebiet der Suchverarbeitung ist somit ein weiterer charakteristischer Vorteil, der sich aus den nachfolgend beschriebenen Systemen und Vorrichtungen ergibt. Since the contextual information can be provided without query, this leads to fewer input errors and erroneously entered queries. Accordingly, in the overall view, thousands of erroneous and inaccurate queries are avoided, which in turn allows more effective use of search system resources. In other words, multiple erroneous query processing cycles are avoided, which reduces the required processing resources and reduces overall system bandwidth requirements (or, alternatively, allows for the operation of a larger number of users without a corresponding increase in processing resources). This improvement in the art of search processing is thus another characteristic advantage resulting from the systems and devices described below.
