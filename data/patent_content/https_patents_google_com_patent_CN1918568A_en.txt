Abstract
A search engine may perform a search for a user search query over a number of possible search categories. For example, the search query may be performed for generai web documents, images, and news documents. The search engine ranks categories based on the searrh query and/or the documents returned for each category and presents the search results to the user by category. Higher ranking categories may be presented more observably than lower ranking categories.

Background technology
WWW (" web ") contains a plurality of information.Search engine is by classification helps the user to locate desired message part to network documentation.Typically, quoting of this request relevant documentation returned in search engine response user's request.
Search engine can be based on the definite judgement for user interest of the search (being called as search inquiry) of user's input.The target of search engine is the link based on search inquiry identification high-quality correlated results.Typically, search engine is finished this operation by clauses and subclauses in the match search inquiry and the network documentation complete or collected works that prestore.The network documentation that includes the user search clauses and subclauses is considered to " hitting " and is returned to the user.
The complete or collected works of the network documentation that prestores can be by the form storage of search engine with the index of the clauses and subclauses found in the network documentation.Document in the index can be divided into one or more classification.Such other example comprises general web documents, commercial product, image, sponsored advertisements/links, and news documents.Classification divided in index can be of value to and help the user to find the result more relevant with their search inquiry.For example, to seeking specific model and the price that the digital camera users interest that will buy may the special concern digital camera, and can focus attentions equally on and the relevant information of digital photograph shooting.
The problem relevant with the search engine that can carry out multi-class search is how to represent different classes of option to the user.A kind of known technology is to allow the user to extract the classification of expection and then inputted search inquiry.For example, the user can at first select " news " label to import the search inquiry that is applied to the news documents index then from the homepage of search engine.Although this technology gives the user a plurality of controls in the guiding search inquiry, the practical problems that this technology runs into is that most users tend to the ignore class distinguishing label, causes their search inquiry to be directed in the default classification.
