Abstract
The present invention relates to the method and system of machine translation feedback, comprising: receive one or more words that will be translated as the first language of second language; Obtain the cypher text of described second language based on described one or more word, translation model and language model, described cypher text is the translation of described one or more word; There is provided graphic user interface (GUI) for showing at display device from described computing equipment, described graphical user interface comprises described one or more word of described first language and described cypher text; Receive user's input of the alternative translation of the described one or more word of instruction; Analyze described user input to determine whether described user input is SPAM; When described user's input is not SPAM, inputs based on described user and upgrade described language model and described translation model; And when described user's input is SPAM, abandon described user input.
