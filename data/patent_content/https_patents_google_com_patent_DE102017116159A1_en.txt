Abstract
Verfahren, Systeme und Vorrichtungen, die Computerprogramme enthalten, die in einem Computerspeichermedium codiert sind, um Kontextinformationen für eine Benachrichtigung bereitzustellen. In einem Aspekt enthält ein Verfahren das Empfangen einer abfrageunabhängigen Anforderung für Kontextinformationen, die für die Benachrichtigung relevant sind, von einer Nutzereinrichtung in Reaktion darauf, dass die Nutzereinrichtung in Reaktion auf ein Auftreten eines Auslöseereignisses eine Benachrichtigung erhält, das Identifizieren mehrerer Suchbegriffe aus dem Inhalt in der Benachrichtigung, das Bestimmen einer Relevanzbewertung für jeden der mehreren Suchbegriffe, das Auswählen eines oder mehrerer der mehreren Suchbegriffe auf der Grundlage der Relevanzbewertungen und das Bereitstellen eines jeweiligen Nutzerschnittstellenelements für die Anzeige mit der Benachrichtigung für die Nutzereinrichtung für jeden der ausgewählten einen oder mehreren mehreren Suchbegriffe, wobei jedes Nutzerschnittstellenelement Kontextinformationen hinsichtlich des jeweiligen Suchbegriffs enthält.A method, systems and apparatus containing computer programs encoded in a computer storage medium to provide context information for notification. In one aspect, a method includes receiving a query-independent request for context information relevant to the notification from a user device in response to the user device receiving a notification in response to an occurrence of a trigger event, identifying multiple search terms from the content in notifying, determining a relevance score for each of the plurality of search terms, selecting one or more of the plurality of search terms based on the relevance scores, and providing each user interface item for the ad with the user facility notification for each of the selected one or more search terms Each user interface element contains context information regarding the respective search term.

HINTERGRUNDBACKGROUND
Diese Patentschrift bezieht sich auf das Bereitstellen von Kontextinformationen für einen Nutzer.This patent relates to providing context information to a user.
Eine Einrichtung kann für einen Nutzer Kontextinformationen bereitstellen. Zum Beispiel kann eine Einrichtung eine Webseite über einen bestimmten Gegenstand anzeigen, eine Suchabfrage von dem Nutzer, die Suchausdrücke für den bestimmten Gegenstand enthält, empfangen, in Reaktion auf die Suchabfrage Suchergebnisse auslesen und die Suchergebnisse für den Nutzer bereitstellen.A device may provide context information to a user. For example, an entity may display a webpage about a particular item, receive a search query from the user containing search terms for the particular item, retrieve search results in response to the search query, and provide the search results to the user.
Typische Interaktionsmodelle erfordern, dass Nutzer eine Form einer Nutzerabfrage für eine Nutzereinrichtung bereitstellen. Zum Beispiel kann ein Nutzer eine Benachrichtigung eingestellt haben, um daran erinnert zu werden, wenn ein Film in den Filmtheatern anlaufen wird. Auf den Tag vor dem Kinostart kann auf dem Smartphone des Nutzers eine Benachrichtigung "Der Film X hat morgen seinen Kinostart" erscheinen. Daraufhin kann sich der Nutzer entscheiden, Filmrezensionen zu prüfen, um zu bestimmen, ob er sich den Film ansehen möchte. Dementsprechend kann der Nutzer äußern: "Telephon, zeig mir Rezensionen für den Film X". Ein Suchprozess führt daraufhin unter Verwendung der Abfrage "Rezensionen für den Film X" eine Suche nach Betriebsmitteln aus, um Betriebsmittel zu identifizieren, die die Informationsnotwendigkeiten des Nutzers erfüllen können.Typical interaction models require that users provide some form of user query for a user device. For example, a user may have set a notification to be reminded when a movie is about to start in the movie theaters. On the day before the theatrical release on the user's smartphone a notification "The film X has its theatrical release tomorrow" appear. The user may then decide to review movie reviews to determine if he wants to watch the movie. Accordingly, the user can express: "Phone, show me reviews for the movie X". A search process then performs a resource search using the "Reviews for Movie X" query to identify resources that may satisfy the user's information needs.

B., wenn der Nutzer nur eine Hand frei hat.Certain embodiments of the subject matter described in this specification may be implemented to realize one or more of the following advantages. When a notification is received, a device may provide context information to a user that is likely to satisfy a user's information needs without a user providing a query to the device or even requesting context information when viewing the notification. This results in a convenient way for the user to obtain context information for notifications. The user no longer needs to enter query expressions or speak query expressions to obtain contextual information. In fact, when the user receives the notification, he may receive context information without providing any indication that the user desires context information for the notification. Accordingly, users receive context information in a manner that is convenient not only for the user, but also in a relatively discrete manner so that bystanders are not disturbed by the user speaking to the device. Further, since the user need not enter a query or need to provide any indication that context information is desired, in some implementations the user may receive context information if the user were not actually able to effectively enter or provide the indication, e.g. For example, if the user has only one hand free.
Da die Kontextinformationen ohne Eingabe von dem Nutzer bereitgestellt werden können, braucht das System außerdem keine Text-zu-Sprache-Verarbeitung auszuführen oder Tastatureingabe zu verarbeiten. Dies führt zu weniger Eingabefehlern und fehlerhaften Eingabeabfragen. Dementsprechend werden insgesamt gesehen tausende fehlerhafter und ungenauer Abfragen vermieden, was wiederum zu einer effizienteren Verwendung von Suchsystembetriebsmitteln führt. Mit anderen Worten, es werden mehrere fehlerhafte Abfrageverarbeitungszyklen vermieden, was die erforderlichen Verarbeitungsbetriebsmittel verringert und die Gesamtsystembandbreiteanforderungen verringert (oder alternativ ermöglicht, dass eine größere Anzahl von Nutzern ohne eine entsprechende Erhöhung der Verarbeitungsbetriebsmittel bedient werden). Diese Verbesserung auf dem technischen Gebiet der Suchverarbeitung ist somit ein weiterer deutlicher Vorteil, der durch die hier beschriebenen Systeme und Verfahren verwirklicht wird.Since the context information can be provided without input from the user, In addition, the system need not perform text-to-speech processing or handle keyboard input. This leads to fewer input errors and erroneous input queries. Accordingly, as a whole, thousands of erroneous and inaccurate queries are avoided, which in turn leads to a more efficient use of search system resources. In other words, multiple erroneous query processing cycles are avoided, which reduces the required processing resources and reduces the overall system bandwidth requirements (or, alternatively, allows a larger number of users to be serviced without a corresponding increase in processing resources). This improvement in the art of search processing is thus another distinct advantage realized by the systems and methods described herein.
