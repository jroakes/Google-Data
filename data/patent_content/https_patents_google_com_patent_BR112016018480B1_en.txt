Abstract
MÉTODO IMPLANTADO POR COMPUTADOR PARA CONFIGURAR UM APLICATIVO, MÍDIA LEGÍVEL POR COMPUTADOR NÃO TRANSITÓRIA E DISPOSITIVO DE COMPUTAÇÃO. Trata-se de um gerenciamento de ajustes de usuário com o uso de fontes externas, que inclui fornecer uma interface de usuário para que um usuário configure um ou mais ajustes que afetam funções de um aplicativo; a interface de usuário permite que os um ou mais ajustes sejam manipulados pelo usuário e permite que os um ou mais ajustes sejam configurados com base em dados de ajuste de uma entidade de terceiros; receber a entrada do usuário para configurar pelo menos uma porção dos um ou mais ajustes com base nos dados de ajuste da entidade; identificar que os dados de ajuste da entidade incluem pelo menos a porção dos um ou mais ajustes; e configurar pelo menos a porção dos um ou mais ajustes com base nos dados de ajuste da entidade.COMPUTER-DEPLOYED METHOD FOR CONFIGURING AN APPLICATION, NON-TRANSIENT COMPUTER READABLE MEDIA, AND COMPUTING DEVICE. This is management of user settings using external sources, which includes providing a user interface for a user to configure one or more settings that affect functions of an application; the user interface allows the one or more settings to be manipulated by the user and allows the one or more settings to be configured based on setting data from a third party entity; receiving user input to configure at least a portion of the one or more adjustments based on the entity's adjustment data; identify that the entity's adjustment data includes at least a portion of the one or more adjustments; and configuring at least a portion of the one or more fits based on the entity fit data.
Description
[001] A matéria discutida no presente documento refere- se, de uma forma geral, a aplicativos de software e, mais particularmente, a gerenciamento de ajustes de usuário com o uso de fontes externas.[001] The matter discussed in this document refers, in general, to software applications and, more particularly, to managing user settings using external sources.
[002] A maioria, se não todos, dos aplicativos com base na web e aplicativos com base em dispositivo (por exemplo, aplicativos que executam em dispositivos móveis e dispositivos que não são móveis, tal como computadores, máquinas, etc.) são projetados para permitir que usuários configurem os aplicativos com o uso de parâmetros ou ajustes. Por exemplo, um aplicativo pode usar um ajuste (por exemplo, “Conectar-se Automaticamente à rede Wi-Fi”) para permitir que um usuário Alice configure ou “diga” ao aplicativo se Alice prefere que o aplicativo conecte automaticamente a uma rede Wi-Fi caso uma esteja disponível (por exemplo, por marcar uma caixa ou inserir “Sim”) ou deixe que Alice inicie manualmente uma conexão (por exemplo, por deixar uma caixa desmarcada ou inserir “Não”). O aplicativo ou outro aplicativo pode usar um ajuste (por exemplo, “Comentador Anônimo”) para permitir que Alice indique sua preferência de privacidade, tal como se deve fornecer comentários para uma plataforma online como “Alice” ou como “Anônimo”.[002] Most, if not all, web-based applications and device-based applications (for example, applications that run on mobile devices and non-mobile devices such as computers, machines, etc.) are designed to allow users to configure applications using parameters or tweaks. For example, an app can use a tweak (for example, “Automatically connect to Wi-Fi network”) to allow an Alice user to configure or “tell” the app whether Alice prefers the app to automatically connect to a Wi-Fi network. -Fi if one is available (for example, by checking a box or entering “Yes”) or let Alice manually initiate a connection (for example, by leaving a box unchecked or entering “No”). The application or other application may use a setting (for example, “Anonymous Commenter”) to allow Alice to indicate her privacy preference, such as whether to provide comments to an online platform as “Alice” or as “Anonymous”.
[003] Um usuário usa, tipicamente, muitos aplicativos ou produtos em mais de um dispositivo, e muitos dos aplicativos são conectados à Internet. Muitos usuários podem não estar cientes de um aplicativo e suas funções em um nível que permite aos mesmos configurar, de forma significativa, os ajustes do aplicativo. Ainda, usuários devem tomar decisões para configurar ajustes de muitos aplicativos que abrangem múltiplos domínios. Essas decisões podem ter impactos tanto de curto quanto de longo prazo.[003] A user typically uses many applications or products on more than one device, and many of the applications are connected to the Internet. Many users may not be aware of an application and its functions to a level that allows them to meaningfully configure application settings. Also, users must make decisions to configure settings for many applications that span multiple domains. These decisions can have both short-term and long-term impacts.
[004] Frequentemente, há dúzias de opções diferentes para ajustes de usuário em um aplicativo ou produto. Cada desses ajustes muda o modo em que um produto funciona em um modo único. Alguns usuários, particularmente aqueles sem experiência em programação de software, podem ter uma grande dificuldade em saber quais opções escolher para um ou mais dados ajustes, de modo que os ajustes tenham suas experiências desejadas. Adicionalmente, muitas dessas opções de ajuste são ad-hoc adicionado e removido (por exemplo, por atualizações de aplicativo). Como resultado, os usuários devem “observar” continuamente seus ajustes para garantir que seus ajustes desejados permaneçam efetivos.[004] There are often dozens of different options for user tweaks in an application or product. Each of these tweaks changes the way a product works in a unique way. Some users, particularly those with no software programming experience, may have great difficulty knowing which options to choose for one or more given tweaks, so that the tweaks have their desired experience. Additionally, many of these tuning options are added and removed ad-hoc (for example, by application updates). As a result, users must continually “watch” their adjustments to ensure that their desired adjustments remain in effect.
