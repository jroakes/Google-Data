Abstract
Computerlesbares Speichermedium mit Anweisungen, die bei Ausführung durch einen oder mehrere Prozessoren Operationen zur Schulung eines Modells für maschinelles Lernen zur Identifikation von Bildbestandteilen in zusammengesetzten Bilder implementieren, wobei die Operationen das Folgende umfassen: Erzeugen von Trainingsdaten für das Modell für maschinelles Lernen, wobei das Erzeugen der Trainingsdaten das Folgende umfasst: Erzeugen eines zusammengesetzten Bildes, umfassend einen ersten Pixeldaten enthaltenden Abschnitt eines ersten Bildbestandteils und einen zweiten Pixeldaten enthaltenden Abschnitt eines zweiten Bildbestandteils; Erzeugen einer ersten Trainingseingabe, die Pixeldaten des zusammengesetzten Bildes enthält; und Erzeugen einer ersten Trainingsausgabe für die erste Trainingseingabe, wobei die erste Trainingsausgabe eine Position des ersten Abschnitts innerhalb des zusammengesetzten Bildes kennzeichnet; und Bereitstellen der Trainingsdaten zur Schulung des Modells für maschinelles Lernen zu (i) einem Satz von die erste Trainingseingabe umfassenden Trainingseingaben und (ii) einem Satz von die erste Zielausgabe umfassenden Zielausgaben.A computer-readable storage medium having instructions that, when executed by one or more processors, implement machine learning model training operations for identifying image components in composite images, the operations comprising: generating training data for the machine learning model, wherein the generating the training data comprises: generating a composite image comprising a first pixel data-containing portion of a first image component and a second pixel data-containing portion of a second image component; Generating a first training input containing pixel data of the composite image; and generating a first training output for the first training input, the first training output identifying a position of the first portion within the composite image; and providing the training data for training the machine learning model on (i) a set of training inputs comprising the first training input and (ii) a set of target outputs comprising the first target output.

AUSFÜHRLICHE BESCHREIBUNGDETAILED DESCRIPTION
Es gibt Situationen mit dem Wunsch, zwei Bilder oder Bildreihenfolgen miteinander zu vergleichen. Während es prinzipiell möglich ist, die zwei Bilder oder Bildreihenfolgen direkt zu vergleichen, ist das wahrscheinlich rechenintensiv und/oder langsam, daher ist es üblich, Versionen eines Bildes oder einer Bildreihenfolge mit niedrigerer Auflösung zu erzeugen, die manchmal als „Fingerabdruck“ des Bildes oder der Bildreihenfolge bezeichnet werden, um diese für den Bildvergleich zu verwenden. Als ein Beispiel kann ein Benutzer einer Content-Sharing-Plattform Inhalte zur Aufnahme durch die Plattform hochladen (z. B. Videos, Bilder, Audioclips usw.), was anderen Benutzern die Möglichkeit gibt, diese Inhalte zu konsumieren (z. B. ansehen o. Ä.).
