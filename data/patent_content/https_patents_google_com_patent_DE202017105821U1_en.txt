HINTERGRUND BACKGROUND
Einige Rechenvorrichtungen (z. B. Mobiltelefone, Tablet-Computer, computerisierte Uhren usw.) können mit Anwendungen geliefert werden, die vorinstalliert sind, oder können es den Benutzern erlauben, Anwendungen aus einem Anwendungs-Depot zu installieren. Einige Anwendungen können es einem Benutzer ermöglichen, verschiedene Anwendungseinstellungen zu ändern. Solche Anwendungen können eine Liste von Anwendungseinstellungen ausgeben, um dem Benutzer zu ermöglichen, verschiedene Einstellungen zu editieren. Die Liste von Anwendungseinstellungen kann für alle Benutzer der Anwendung gleich sein. Some computing devices (eg, cell phones, tablet computers, computerized watches, etc.) may be shipped with applications that are preinstalled, or may allow users to install applications from an application depot. Some applications may allow a user to change various application settings. Such applications can output a list of application settings to allow the user to edit various settings. The list of application settings may be the same for all users of the application.

ZUSAMMENFASSUNG SUMMARY
Der offenbarte Gegenstand bezieht sich auf spezielle Techniken zum dynamischen Erzeugen und Anzeigen benutzerspezifischer Gruppen von Anwendungseinstellungen. Eine Anwendung, die auf einer Rechenvorrichtung abläuft, kann bestimmen, welche Einstellungen aus einer Gruppe von Einstellungen, die ein Benutzer der Anwendung am wahrscheinlichsten auswählt, basierend auf Informationen, die einem Benutzer der Rechenvorrichtung zugeordnet sind. Die Informationen können durch die Rechenvorrichtung nur in Fällen verwendet werden, in denen der Benutzer explizite Autorisierung für die Rechenvorrichtung bereitstellt, auf solche Informationen zuzugreifen und sie zu benutzen. In Reaktion auf das Bestimmen der Einstellungen, die am wahrscheinlichsten ausgewählt werden, kann die Anwendung eine benutzerspezifische graphische Benutzerschnittstelle erzeugen, die die Einstellungen, die am wahrscheinlichsten ausgewählt werden, prominenter anzeigt als andere Einstellungen. Auf diese Weise können die Techniken dieser Offenbarung eine Gruppe von Einstellungen bereitstellen, die für den Benutzer der Vorrichtung individuell zugeschnitten sind. Die Techniken der Offenbarung können automatisch und dynamisch ermöglichen, dass die Einstellungen, die durch den Benutzer am wahrscheinlichsten ausgewählt werden, an einer prominenteren Position angezeigt werden, und dem Benutzer ersparen, dass er jedes Mal alle Einstellungen durchsuchen und die interessanten Einstellungen auswählen muss. Das trägt dazu bei, die Benutzerschnittstelle zu vereinfachen. Eine solche Vereinfachung ist insbesondere nützlich und vorteilhaft auf Rechenvorrichtungen wie z. B. mobilen oder tragbaren Vorrichtungen, die typischerweise eine begrenzte Anzeigegröße und/oder Benutzerinteraktion aufweisen. The disclosed subject matter relates to specific techniques for dynamically generating and displaying user-specific sets of application settings. An application running on a computing device may determine which settings from a set of settings a user of the application most likely selects based on information associated with a user of the computing device. The information may be used by the computing device only in cases where the user provides explicit authorization for the computing device to access and use such information. In response to determining the settings that are most likely to be selected, the application may generate a user-specific graphical user interface that displays the settings that are most likely to be selected more prominently than other settings. In this way, the techniques of this disclosure may provide a set of settings that are customized for the user of the device. The techniques of the disclosure may automatically and dynamically enable the settings most likely to be selected by the user to be displayed in a more prominent position and save the user from having to search all settings and select the interesting settings each time. This helps to simplify the user interface. Such a simplification is particularly useful and advantageous to computing devices such. Mobile or portable devices, which typically have limited display size and / or user interaction.

Die Techniken dieser Offenbarung können es ermöglichen, dass eine Anwendung vorhersagt, welche Einstellungskategorien am wahrscheinlichsten durch einen Benutzer der Anwendung auswählt werden, und eine graphische Darstellung der vorhergesagten Einstellungskategorien an prominenteren Orten der graphischen Benutzerschnittstelle ausgeben. B. mobilen oder tragbaren Vorrichtungen, die typischerweise eine begrenzte Anzeigegröße und/oder Benutzerinteraktion aufweisen. This helps to simplify the user interface. Such a simplification is particularly useful and advantageous to computing devices such. Mobile or portable devices, which typically have limited display size and / or user interaction.
