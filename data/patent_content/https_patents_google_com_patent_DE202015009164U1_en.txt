Typically, geographic software provides directional, rotation, and zooming controls for positioning a window, which can be called a visible portion of the digital map in a "viewport" over the desired point. These controls can overlay the digital map in the form of buttons, for example. As another example, geographic software on a touchscreen device can support user gestures so that the user can swipe in the desired direction by swiping his or her finger over the digital map, increasing an area by contracting two fingers, or expanding the area by pulling apart can reduce two fingers.

Überblickoverview
Eine geografische Anwendung erkennt die kontinuierliche Bewegung eines Fensters, in dem eine digitale Karte angezeigt wird („Karten-Ansichtsfenster”), ermittelt einen geografischen Ort, der innerhalb des Ansichtsfensters sichtbar wird, wenn sich das Ansichtsfenster weiterhin entlang seines aktuellen Bewegungsverlaufs bewegt, und stellt ein Benutzersteuerelement zur direkten Platzierung des Ansichtsfensters über den geografischen Ort zur Verfügung. Somit kann das Ansichtsfenster schnell an eine neue Position nach Bestätigung durch den Benutzer „springen”. Je nach Implementierung bzw. Szenario kann die geografische Anwendung ein einzelnes Ziel oder mehrere Ziele erkennen und vorschlagen, zu dem/denen das Ansichtsfenster springen kann.A geographic application detects the continuous movement of a window displaying a digital map ("map viewport"), detects a geographic location that becomes visible within the viewport as the viewport continues to move along its current motion history a user control to directly place the viewport over the geographic location. Thus, the viewport can quickly "jump" to a new position upon user confirmation. Depending on the implementation or scenario, the geographic application can detect and suggest a single target or targets to which the viewport may jump.
In einem exemplarischen Szenario betrachtet der Benutzer eines Touchscreen-Gerätes, wie einem Smartphone, einer Karte von Los Angeles, Kalifornien. Die Zoomstufe ist so eingestellt, dass ein Großteil der Stadt auf den Bildschirm passt. Der Benutzer streicht dann über die Karte, um das Ansichtsfenster in einer allgemeinen nordwestlichen Richtung zu bewegen. Das Ansichtsfenster kann sich dann weiterbewegen, um etwa über Santa Clarita ausgerichtet zu werden. Die Benutzer streicht erneut über den Bildschirm, und das Ansichtsfenster wird nun etwa über den östlichen Teil des Los Padres National Forest zentriert. Nach einem weiteren Streichen zentriert sich das Ansichtsfenster etwa über das Bitter Creek National Wildlife Refuge. Die geografische Anwendung könnte durch die drei aufeinanderfolgenden Streichungen in ähnlicher Richtung eine bedeutende kontinuierliche Bewegung des Ansichtsfensters definieren, sollte der Benutzer weiterhin in diese Richtung streichen, und würde das Ansichtsfenster dem wahrscheinlichen Bewegungsverlauf folgen, feststellen, dass der Bewegungsverlauf San Francisco, CA durchläuft und ein Fenster bereitstellen, dass die digitale Karte überlagert und das einen Link zu San Francisco, CA enthält. Falls der Benutzer auf den Link klickt, positioniert die geografische Anwendung das Ansichtsfenster direkt über San Francisco.In an exemplary scenario, the user of a touchscreen device, such as a smartphone, views a map of Los Angeles, California. The zoom level is set to fit most of the city on the screen. The user then strokes the map to move the viewport in a general northwest direction. The viewport can then move on, for example, to be aligned over Santa Clarita. The user strokes the screen again, and the viewport is now centered over the eastern part of the Los Padres National Forest. After another stroke, the viewport centers around the Bitter Creek National Wildlife Refuge. The geographic application could define a significant continuous movement of the viewport by the three consecutive strokes in the similar direction should the user continue to swipe in that direction, and would the viewport follow the likely course of motion, notice that the motion traverses and passes through San Francisco, CA. Provide windows overlaying the digital map and containing a link to San Francisco, CA. If the user clicks the link, the geographic application positions the viewport directly over San Francisco.
Ganz allgemein kann die geografische Anwendung ermitteln, ob das Ansichtsfenster im Wesentlichen kontinuierlich durch die Ermittlung einer bestimmten Anzahl von aufeinanderfolgenden Streichungen in einer ähnlichen Richtung, einer bestimmten Anzahl von aufeinanderfolgenden Mausklicks, einer bestimmten Anzahl von aufeinanderfolgenden Sprachbefehlen usw. bewegt wird. Des Weiteren kann die geografische Anwendung in einer Gestenbasierten Umgebung im Wesentlichen kontinuierliche Bewegungen des Ansichtsfensters als Reaktion auf eine kleinere Anzahl von aufeinanderfolgenden Streichungen identifizieren, wenn die Beschleunigung, mit der der Benutzer mit dem Ansichtsfenster mittels Streichgesten kommuniziert, einen bestimmten Schwellenwert überschreitet.More generally, the geographic application may determine whether the viewport is moved substantially continuously by determining a certain number of consecutive strokes in a similar direction, a certain number of consecutive mouse clicks, a certain number of consecutive voice commands, and so on. Further, in a gesture-based environment, the geographic application may identify substantially continuous views of the viewport in response to a smaller number of consecutive strokes when the acceleration with which the user communicates with the viewport via swipe gestures exceeds a certain threshold.
Die geografische Anwendung kann ein oder mehrere Ansichtsziele im Hinblick auf die Zoomstufe sowie die aktuelle Bewegungsrichtung des Ansichtsfensters identifizieren. So kann die geografische Anwendung beispielsweise ein weiter entferntes Ziel identifizieren, wenn sich das Ansichtsfenster auf einer relativ niedrigen Zoomstufe befindet (und daher eine relativ große geografische Fläche abgedeckt wird) und ein weniger weit entferntes Ziel, wenn das Ansichtsfenster sich auf einer relativ hohen Zoomstufe befindet (und daher eine relativ kleine Fläche abgedeckt wird). The geographic application can identify one or more view targets based on the zoom level and the current direction of movement of the viewport. For example, the geographic application may identify a farther target if the viewport is at a relatively low zoom level (thus covering a relatively large geographic area) and a less distant target if the viewport is at a relatively high zoom level (and therefore a relatively small area is covered).
