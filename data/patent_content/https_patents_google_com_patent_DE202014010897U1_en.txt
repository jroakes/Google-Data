Abstract
Ein nicht transitorisches computerlesbares Medium, das darin gespeicherte Anweisungen umfasst, die, wenn sie von einem Prozessor ausgeführt werden, den Prozessor dazu veranlassen, ein Verfahren durchzuführen, das unter anderem das Folgende umfasst: das Bestimmen der benachbarten Zugriffspunkte aus Scans von Netzwerk-Zugriffspunkten in einem Raum, wobei jeder benachbarte Zugriffspunkt zusammen in einem Scan einer bestimmten Ebene des Raumes erfolgt; das Empfangen von drahtlosen Daten aus einer Vielzahl von mobilen Geräten, die sich durch einen Raum bewegen; das Identifizieren eines Satzes alles Zugriffspunkte für den Raum basierend auf den drahtlosen Daten; und das Ableiten eines Verhältnisses unter Verwendung eines Prozessors, bezogen auf einen Unterschied zwischen den benachbarten Zugriffspunkten und dem Satz aller Zugriffspunkte, wobei das Verhältnis einen Prozentsatz der fehlenden Zugriffspunkte für die bestimmte Ebene des Raumes darstellt.A non-transitory computer readable medium having instructions stored therein that, when executed by a processor, cause the processor to perform a method comprising, among other things: determining the adjacent access points from scans of network access points in a space, each adjacent access point being together in a scan of a particular level of the space; receiving wireless data from a plurality of mobile devices moving through a room; identifying a set of all access points for the room based on the wireless data; and deriving a ratio using a processor based on a difference between the adjacent access points and the set of all access points, the ratio representing a percentage of the missing access points for the particular level of the space.

For example, a location in an interior may be determined by comparing data from wireless access points that a user's mobile device receives to a database in which mappings are stored between the wireless access points and the indoor locations.
Um diese Art von Datenbank zu erstellen, wird ein Innenraumstandort-Anbieter in der Regel Positionserkundungen an ausgewählten Standorten durchführen. Mit der Zeit können einige drahtlose Zugriffspunkte jedoch wegfallen, verlegt werden oder anderweitig funktionslos werden, was zu falschen Positionsvorhersagen und zu zunehmenden Fehlern bei der Positionsvorhersage führt.To create this type of database, an indoor location provider will typically perform position surveys at selected locations. Based on these explorations, a model of the interior is then created by searching for positions of wireless access points and the corresponding signal strengths at these locations. However, over time, some wireless access points may be dropped, misplaced, or otherwise rendered inoperative, resulting in incorrect position predictions and increasing position prediction errors.

Diese Daten können beispielsweise durch Scannen auf drahtlose Zugriffspunkte während einer Standorterfassung des Raumes gesammelt werden. The wireless data may include information such as network access point IDs such as a MAC address and / or SSIDs, as well as signal strengths. For example, this data may be collected by scanning for wireless access points during a location survey of the room. The wireless data from the scans defines adjacent access points for a given level. Each adjacent access point appears together in a single scan that is used to create a radio network model for that particular plane.
Unterbrechungen im Funknetzmodell können durch Vergleich der benachbarten Zugriffspunkte mit den Zugriffspunktdaten erkannt werden, die von mehreren Benutzern, die sich durch den Raum bewegen, erfasst werden. Unterbrechungen im Funknetzmodell) für die jeweilige Ebene des Innenraums dar.Interrupts in the radio network model can be detected by comparing the neighboring access points with the access point data captured by multiple users moving through the room. Based on this analysis, it can be determined whether the neighboring access points have moved or been dropped. In this regard, a ratio can be calculated to determine if the neighboring access points have been dropped. For example, this ratio may be based on a difference that exists between the access points used to create the particular level radio network model and a set of all the access points that users gather through crowdsourcing at different levels in space over time. The ratio represents a percentage of missing network access points (e.g., breaks in the radio network model) for each level of the interior.
