Description
무선 통신 기술의 확산으로 인해 무선 네트워크를 통해 원격 데이터 또는 기타 서비스에 액세스할 수 있는 디바이스의 수가 크게 증가했다. 셀룰러 네트워크와 같은 광역 네트워크의 경우, 코어 네트워크 운영자는 일반적으로 네트워크를 통해 이동국에 통신 서비스를 제공한다. 네트워크의 일 예는 코어 네트워크와 기지국을 포함하는 PLMN(Public Land Mobile Network)이다. 기지국은 집합적으로 네트워크에 연결된 무선 접속 네트워크(RAN)를 형성한다. 네트워크와 무선으로 통신하기 위해, 이동국은 PLMN의 셀(예를 들어, 기지국, 서빙 셀)을 통해 코어 네트워크에 대한 연결을 설정할 수 있다.The proliferation of wireless communication technologies has significantly increased the number of devices that can access remote data or other services over wireless networks. In the case of a wide area network, such as a cellular network, a core network operator generally provides communication services to mobile stations via the network. An example of the network is a Public Land Mobile Network (PLMN) including a core network and a base station. The base stations collectively form a radio access network (RAN) connected to the network. In order to communicate wirelessly with the network, the mobile station may establish a connection to the core network via a cell (eg, a base station, a serving cell) of the PLMN.
이 연결을 설정하기 위한 ATTACH 절차 동안, 네트워크는 이동국에 의해 지원되는 무선 통신 능력(capabilities)에 관해 문의할 것이다. PLMN과의 성공적인 연결을 위해, 이동국은 지원되는 주파수 대역 및 기타 무선 통신 파라미터를 포함하는 능력으로 네트워크에 응답해야 한다. 증가된 데이터 속도를 위해, 일부 이동국 및 네트워크는 이동국이 다수의 채널을 사용하여 네트워크와 데이터를 통신할 수 있도록 하는 반송파 집성(CA)을 구현한다. 반송파 집성을 가능하게 하기 위해, 이동국은 또한 무선 통신을 위한 기능의 일부로 반송파, 대역 및 대역폭과 같은 CA 파라미터의 다양한 조합을 네트워크에 전달한다.During the ATTACH procedure to establish this connection, the network will inquire about the wireless communication capabilities supported by the mobile station. For a successful connection with the PLMN, the mobile station must respond to the network with the capability to include supported frequency bands and other wireless communication parameters. For increased data rates, some mobile stations and networks implement carrier aggregation (CA), which allows mobile stations to communicate data with the network using multiple channels. To enable carrier aggregation, the mobile station also conveys various combinations of CA parameters such as carrier, band and bandwidth to the network as part of the function for wireless communication.
그러나, 네트워크가 다중 대역을 갖는 더 많은 CA 가능 셀을 배치함에 따라, 많은 이동국에 의해 지원되는 CA 파라미터 조합의 수 및 크기가 상당히 증가했다. CA 파라미터 조합의 수와 크기가 증가함에 따라, ATTACH 절차 동안 네트워크로 전송되는 이동국의 능력 메시지 크기도 상당히 증가했다. 네트워크 커버리지가 좋지 않거나 신호 세기가 약한 경우, CA 파라미터 조합을 포함하는 기능 메시지의 전송은 네트워크 처리량이 감소하기 때문에 완료하는데 시간이 더 오래 걸리는 경우가 많다. 많은 네트워크 운영자에 의해 구현되는 보호 시간으로 인해, 보호 시간 안에 이동국으로부터 완전한 능력 메시지가 수신되지 않는 경우 ATTACH 절차 동안 이동국과의 연결이 끊어질 수 있다. 추가적으로, 다수의 실패한 ATTACH 절차 후에, 이동국은 일정 시간 동안 후속 ATTACH 절차로부터 PLMN을 차단할 것이다. 이와 같이, 열악한 네트워크 커버리지를 갖는 이동국은 종종 PLMN과의 ATTACH 절차를 완료할 수 없거나, 이동국의 네트워크 서비스가 손상되거나 거부될 수 있는 일정 시간 동안 ATTACH 절차를 재시도하는 것이 차단될 수 있다.However, as the network deploys more CA capable cells with multiple bands, the number and size of CA parameter combinations supported by many mobile stations has increased significantly. As the number and size of CA parameter combinations increases, the size of the mobile station's capability message transmitted to the network during the ATTACH procedure also increases significantly. When the network coverage is poor or the signal strength is weak, the transmission of the function message including the CA parameter combination often takes longer to complete because the network throughput is reduced. Due to the guard time implemented by many network operators, the connection with the mobile station may be lost during the ATTACH procedure if a complete capability message is not received from the mobile station within the guard time. Additionally, after a number of failed ATTACH procedures, the mobile station will block the PLMN from subsequent ATTACH procedures for a period of time. As such, a mobile station with poor network coverage may often be unable to complete the ATTACH procedure with the PLMN or may be blocked from retrying the ATTACH procedure for a period of time during which the mobile station's network service may be compromised or denied.
본 개시는 변화되는(다양한) 네트워크 상태에 대한 적응적 PLMN 관리의 디바이스 및 기술을 설명한다. 일부 양태에서, 사용자 장비(UE)는 등록 절차의 일부로서, 기정의된 지속 시간 동안 등록이 차단된 PLMN을 포함하는 PLMN 정보의 데이터 저장소를 검색하는 단계를 포함하는 방법을 수행한다. 방법은 등록이 차단된 PLMN이 UE와 PLMN의 셀 간의 연결의 신호 관련 상태로 인해 차단된 것으로 결정한다. 방법은 또한 기정의된 지속 시간 동안 UE와 PLMN의 셀 간의 연결의 신호 관련 상태가 개선되었다고 결정하는 단계를 포함한다. 신호 관련 상태의 개선에 응답하여, 방법은 PLMN의 등록 차단을 해제한다. 그런 다음, 방법은 기정의된 지속 시간의 만료 전에 PLMN에 등록할 수 있도록 PLMN의 셀로 차단 해제된 PLMN에 대한 등록 요청을 전송한다.This disclosure describes devices and techniques of adaptive PLMN management for changing (various) network conditions. In some aspects, a user equipment (UE) performs, as part of a registration procedure, a method comprising searching a data store of PLMN information comprising PLMNs that have been blocked from registration for a predefined duration. The method determines that the PLMN whose registration is blocked is blocked due to a signal related state of the connection between the UE and the cell of the PLMN. The method also includes determining that a signal related state of a connection between the UE and a cell of the PLMN has improved for a predefined duration. In response to the improvement of the signal-related state, the method unblocks registration of the PLMN. Then, the method sends a registration request for the unblocked PLMN to the cell of the PLMN so that it can register with the PLMN before the expiration of the predefined duration.
다른 양태에서, 사용자 장비는 PLMN에 대한 제1 등록 요청을 PLMN의 셀로 전송하는 단계를 포함하는 방법을 수행한다. 그런 다음, 방법은 UE 능력 메시지를 전송하는 단계 동안 PLMN에 대한 제1 등록 요청의 실패를 검출한다. 방법은 PLMN의 셀과 통신하기 위한 신호 관련 특성이 임계값을 충족하지 못하는 것으로 결정하는 단계 및 신호 관련 특성에 기초하여 PLMN 등록 실패를 식별하는 단계를 포함한다. 그런 다음 방법은 지속 시간 동안, PLMN의 셀과 통신하기 위한 신호 관련 특성이 동일하게 유지되는지, 개선되는지 또는 악화되는지 여부를 결정한다. 신호 관련 특성의 결정에 응답하여, 방법은 PLMN에 대한 다음 등록 요청 동안 UE 능력 메시지에서 더 적거나 더 많은 수의 CA 조합을 사용한다. 신호 상태가 크게 개선되면, UE는 지속 시간이 만료되기 전에 등록을 위해 다른 시도를 할 수 있다.In another aspect, the user equipment performs a method comprising sending a first registration request for the PLMN to a cell of the PLMN. Then, the method detects failure of the first registration request to the PLMN during the step of sending the UE capability message. The method includes determining that a signal-related characteristic for communicating with a cell of the PLMN does not meet a threshold value and identifying a PLMN registration failure based on the signal-related characteristic. The method then determines, for the duration, whether the signal-related characteristics for communicating with the cell of the PLMN remain the same, improve, or deteriorate. In response to the determination of the signal related characteristic, the method uses a smaller or larger number of CA combinations in the UE capability message during the next registration request to the PLMN. If the signal condition improves significantly, the UE may make another attempt to register before the duration expires.
또 다른 양태에서, 사용자 장비(UE)는 하나 이상의 안테나와 작동 가능하게 결합된 송수신기와, 송수신기와 관련된 하드웨어 기반 프로세서와, 그리고 UE의 하드웨어 기반 프로세서에 의해 실행 가능한 명령들을 저장하는 컴퓨터 판독 가능 저장 매체를 포함한다. 명령들은 등록 절차의 일부로서, 기정의된 지속 시간 동안 등록이 차단된 PLMN을 포함하는 PLMN 정보의 데이터 저장소를 검색하도록 사용자 장비에 지시하도록 구성된 PLMN 관리자를 구현하기 위해 실행될 수 있다. PLMN 관리자는 등록이 차단된 PLMN이 단말과 PLMN의 셀 사이의 연결 신호 관련 상태로 인해 차단된 것으로 결정한다. 그런 다음 PLMN 관리자는 송수신기를 통해 지속 시간 동안, UE와 PLMN의 셀 간의 연결의 신호 관련 상태가 개선되었음을 결정한다. 신호 관련 상태의 개선에 응답하여, PLMN 관리자는 PLMN의 등록을 차단 해제한다. 그런 다음 PLMN 관리자는 송수신기를 통해 PLMN의 기지국으로, 기정의된 지속 시간의 만료 전에 PLMN에 등록할 수 있도록 차단 해제된 PLMN에 대한 등록 요청을 전송한다.In another aspect, a user equipment (UE) includes a transceiver operatively coupled to one or more antennas, a hardware-based processor associated with the transceiver, and a computer-readable storage medium storing instructions executable by the hardware-based processor of the UE. includes The instructions may be executed to implement, as part of a registration procedure, a PLMN manager configured to instruct the user equipment to search a data store of PLMN information including a PLMN that has been blocked from registration for a predefined duration. The PLMN manager determines that the PLMN whose registration is blocked is blocked due to a connection signal-related state between the UE and the cell of the PLMN. Then, the PLMN manager determines through the transceiver that, for the duration, the signal-related state of the connection between the UE and the cell of the PLMN has improved. As part of connection establishment or maintenance, a user equipment (UE) may register with the PLMN via an ATTACH procedure or a tracking area update (TAU). When the UE attempts to register with the PLMN (network), the network will inquire about the UE's wireless communication capabilities. For successful registration, the UE may respond to the network with its wireless communication capability (eg, a UE capability message) enabling communication between the UE and the network. Some UEs and PLMNs provide increased bandwidth through carrier aggregation (CA), which allows the UE to communicate data with the network using multiple channels. To enable carrier aggregation with the network, these UEs also convey various combinations of CA parameters such as carrier, band and bandwidth to the network as part of the UE capability message.

Then, the PLMN manager determines that the signal-related state between the UE and the cell of the PLMN has improved for the duration. In response to the improvement, the PLMN administrator unblocks the PLMN. By doing so, the UE may retry registration with the unblocked PLMN at least once before the duration expires, which allows the UE to register faster and re-establish network services.

일반적으로, 본 명세서에 설명된 컴포넌트, 모듈, 방법 및 동작들은 소프트웨어, 펌웨어, 하드웨어(예를 들어, 고정 논리 회로), 수동 처리 또는 이들의 임의의 조합을 사용하여 구현될 수 있다. 예시적인 방법의 일부 동작들은 컴퓨터 처리 시스템에 대해 로컬 및/또는 원격인 컴퓨터 판독 가능 저장 메모리에 저장된 실행 가능한 명령들의 일반적인 맥락에서 설명될 수 있으며, 구현들은 소프트웨어 애플리케이션, 프로그램, 기능 등을 포함할 수 있다. 대안적으로 또는 추가로, 본 명세서에 설명된 임의의 기능은 이에 한정되지 않지만 FPGA, ASIC, ASSP, SoC, CPLD, 코-프로세서, 컨텍스트 허브, 센서 코-프로세서 등과 같은 하나 이상의 하드웨어 논리 컴포넌트에 의해 적어도 부분적으로 수행될 수 있다.In general, the components, modules, methods, and acts described herein may be implemented using software, firmware, hardware (eg, fixed logic circuitry), manual processing, or any combination thereof. Some operations of the example method may be described in the general context of executable instructions stored in computer-readable storage memory that is local and/or remote to a computer processing system, implementations of which may include software applications, programs, functions, and the like. there is. Alternatively or additionally, any functionality described herein may be implemented by one or more hardware logic components such as, but not limited to, FPGAs, ASICs, ASSPs, SoCs, CPLDs, co-processors, context hubs, sensor co-processors, and the like. at least partially.
사용자 장비(UE)에 의해 수행되는 제1 방법은 등록 절차의 일부로서, 기정의된 지속 시간 동안 등록이 차단된 PLMN을 포함하는 PLMN(Public Land Mobile Network) 정보의 데이터 저장소를 검색하는 단계와; 등록이 차단된 PLMN이 단말과 PLMN의 셀 사이의 신호 관련 상태로 인해 차단되었다고 판단하는 단계와; 기정의된 지속 시간 동안, UE와 PLMN의 셀 간의 연결의 신호 관련 상태가 개선되었다고 결정하는 단계와; 신호 관련 상태의 개선에 응답하여 PLMN의 등록 차단을 해제하는 단계와; 그리고 PLMN의 셀로, 기정의된 지속 시간이 만료되기 전에 PLMN에 등록할 수 있도록 차단 해제된 PLMN에 등록 요청을 전송하는 단계를 포함한다.A first method performed by a user equipment (UE) comprises, as part of a registration procedure, searching a data store of Public Land Mobile Network (PLMN) information containing PLMNs that have been blocked from registration for a predefined duration; determining that the PLMN whose registration is blocked is blocked due to a signal-related state between the terminal and the cell of the PLMN; determining, for a predefined duration, that a signal related state of a connection between the UE and a cell of the PLMN has improved; unblocking registration of the PLMN in response to the improvement of the signal-related state; and sending, to the cell of the PLMN, a registration request to the unblocked PLMN so that it can register with the PLMN before the predefined duration expires.
전술한 제1 방법에 더하여, 사용자 장비에 의해 수행되는 제2 방법은 PLMN의 셀로, PLMN에 대한 제1 등록 요청을 전송하는 단계와; PLMN에 대한 제1 등록 요청의 실패를 검출하는 단계와; 제1 등록 요청의 실패에 응답하여 지속 시간 동안 PLMN의 등록을 차단하는 단계와; PLMN의 셀과 통신하기 위한 신호 관련 특성이 임계값을 충족하지 못하는 것으로 결정하는 단계와; 신호 관련 특성에 기초하여 차단된 PLMN을 식별하는 단계와; 지속 시간 동안, PLMN의 셀과 통신하기 위한 신호 관련 특성이 임계값을 충족한다고 결정하는 단계와; 임계값을 충족하는 신호 관련 특성에 응답하여, 신호 관련 특성에 기초하여 차단된 것으로 식별된 PLMN을 차단 해제하는 단계와; 그리고 PLMN의 셀로, 지속 시간의 만료 전에 PLMN에 대한 제2 등록 요청을 전송하는 단계를 포함한다.In addition to the first method described above, a second method performed by the user equipment includes: sending, to a cell of the PLMN, a first registration request for the PLMN; detecting failure of the first registration request to the PLMN; blocking registration of the PLMN for a duration in response to the failure of the first registration request; determining that a signal related characteristic for communicating with a cell of the PLMN does not meet a threshold value; identifying a blocked PLMN based on the signal related characteristic; determining, for a duration, that a signal related characteristic for communicating with a cell of the PLMN meets a threshold; in response to the signal-related characteristic meeting the threshold, unblocking the PLMN identified as blocked based on the signal-related characteristic; and sending, to the cell of the PLMN, a second registration request for the PLMN before the expiration of the duration.
전술한 방법에 추가하여, 사용자 장비에 의해 수행되는 제3 방법은 PLMN에 대한 제1 등록 요청을 PLMN의 셀에 전송하는 단계와; UE 능력 메시지를 전송하는 단계 동안 PLMN에 대한 제1 등록 요청의 실패를 검출하는 단계와; PLMN의 셀과 통신하기 위한 신호 관련 특성이 임계값을 충족하지 못하는 것으로 결정하고 신호 관련 특성에 기초하여 PLMN 등록 실패를 식별하는 단계와; 일정 지속 시간 동안, PLMN의 셀과 통신하기 위한 신호 관련 특성이 동일하게 유지되는지, 개선되는지 또는 악화되는지 여부를 결정하는 단계와; 그리고 신호 관련 특성의 결정에 응답하여, PLMN에 대한 다음 요청 동안 UE 능력 메시지에서 더 적거나 더 많은 수의 반송파 집성 조합을 사용하는 단계를 포함한다.In addition to the above method, a third method performed by the user equipment includes: sending a first registration request for the PLMN to a cell of the PLMN; detecting failure of the first registration request to the PLMN during the step of sending the UE capability message; determining that a signal-related characteristic for communicating with a cell of the PLMN does not meet a threshold and identifying a PLMN registration failure based on the signal-related characteristic; determining, for a period of time, whether a signal-related characteristic for communicating with a cell of the PLMN remains the same, improves, or deteriorates; and in response to determining the signal related characteristic, using the smaller or larger number of carrier aggregation combinations in the UE capability message during a next request to the PLMN.
상술한 방법에 더하여, 사용자 장비는 하나 이상의 안테나와 작동 가능하게 결합된 송수신기와; 송수신기와 관련된 하드웨어 기반 프로세서와; 그리고 PLMN 관리자를 구현하기 위한 명령들을 저장하는 컴퓨터 판독 가능 저장 매체를 포함하고, PLMN 관리자는 등록 절차의 일부로서, 기정의된 지속 시간 동안 등록이 차단된 PLMN을 포함하는 PLMN 정보의 데이터 저장소를 검색하고, 등록이 차단된 PLMN이 UE와 PLMN의 셀 간의 연결의 신호 관련 상태로 인해 차단되었다고 결정하고; 송수신기를 통해 기정의된 지속 시간 동안, UE와 PLMN의 셀 간의 연결의 신호 관련 상태가 개선되었다고 결정하고; 신호 관련 상태의 개선에 응답하여 PLMN의 등록 차단을 해제하고; 그리고 송수신기를 통해 PLMN의 셀로, 기정의된 지속 시간의 만료 전에 PLMN에 등록할 수 있도록 차단 해제된 PLMN에 대한 등록 요청을 전송하도록 사용자 장비(UE)에 지시하도록 구성된다.In addition to the method described above, user equipment includes a transceiver operatively coupled to one or more antennas; a hardware-based processor associated with the transceiver; and a computer readable storage medium storing instructions for implementing a PLMN manager, wherein the PLMN manager retrieves, as part of a registration procedure, a data store of PLMN information including a PLMN whose registration has been blocked for a predefined duration. and determine that the PLMN whose registration is blocked is blocked due to a signal related state of a connection between the UE and a cell of the PLMN; determine, for a predefined duration via the transceiver, that the signal related state of the connection between the UE and the cell of the PLMN has improved; unblock registration of the PLMN in response to improvement of the signal-related condition; and instruct the user equipment (UE) to send a registration request for the unblocked PLMN to the cell of the PLMN via the transceiver so that it can register with the PLMN before expiration of a predefined duration.
전술한 방법 및 사용자 장비에 추가하여, UE에 의해 전송된 등록 요청은 PLMN에 대한 ATTACH 요청 또는 PLMN에 대한 트래킹 영역 업데이트 요청을 포함한다.In addition to the above method and user equipment, the registration request sent by the UE includes an ATTACH request to the PLMN or a tracking area update request to the PLMN.
전술한 방법 및 사용자 장비에 추가하여, 등록이 차단된 PLMN은 임시 금지 PLMN 목록에 있고; PLMN의 차단 해제는 임시 금지 PLMN 목록으로부터 PLMN을 차단 해제하고; 그리고 기정의된 지속 시간은 임시 금지 PLMN 목록과 관련된 타이머에 의해 추적된다.In addition to the above-mentioned method and user equipment, the PLMN whose registration is blocked is in the list of temporarily banned PLMNs; unblocking the PLMN unblocks the PLMN from the temporarily banned PLMN list; And the predefined duration is tracked by the timer associated with the temporarily banned PLMN list.
전술한 방법 및 사용자 장비에 추가하여, 신호-관련 상태의 개선을 검출하는 단계는 UE와 셀 간의 연결의 신호 관련 특성을 측정하는 단계와; 그리고 연결의 신호 관련 특성이 임계값을 충족한다고 결정하는 단계를 포함한다. In addition to the method and user equipment described above, detecting an improvement in a signal-related state includes: measuring a signal-related characteristic of a connection between the UE and the cell; and determining that a signal-related characteristic of the connection meets the threshold.
전술한 방법 및 사용자 장비에 추가하여, UE와 셀 간의 연결의 신호 관련 특성은 수신 신호 세기, 수신 신호 품질, 기준 신호 수신 전력, 기준 신호 수신 품질, 수신 신호 세기 표시자, 반송파 대 간섭 비, 신호 대 잡음 비, 비트 오류율 또는 패킷 오류율 중 하나 이상을 포함한다.In addition to the above-described method and user equipment, the signal-related characteristics of the connection between the UE and the cell include: received signal strength, received signal quality, reference signal received power, reference signal received quality, received signal strength indicator, carrier-to-interference ratio, signal includes one or more of a noise-to-noise ratio, a bit error rate, or a packet error rate.
전술한 방법 및 사용자 장비에 추가하여, 등록 절차의 일부로 등록 요청 또는 후속 등록 요청의 실패를 검출하는 단계와; 그리고 기정의된 지속 시간의 나머지 부분 동안 PLMN을 다시 차단하는 단계를 포함한다.In addition to the method and user equipment described above, the method comprising the steps of: detecting, as part of a registration procedure, failure of a registration request or a subsequent registration request; and blocking the PLMN again for the remainder of the predefined duration.
전술한 방법 및 사용자 장비에 추가하여, 등록 요청은 제2 등록 요청이고, 방법은 PLMN에 대한 제1 등록 요청의 실패를 검출하는 단계와; 제1 등록 요청의 실패가 등록 실패의 최대 횟수라고 결정하는 단계와; 제1 등록 요청의 실패가 등록 실패의 최대 횟수인 것에 응답하여, 기정의된 지속 시간 동안 PLMN에 대한 등록을 차단하는 단계와; 제1 등록 요청의 실패가 UE와 PLMN의 셀 간의 연결의 신호 관련 상태로 인한 것이라고 결정하는 단계와; 그리고 PLMN 정보의 데이터 저장소에서, 신호 관련 상태로 인해 차단된 PLMN을 식별하는 단계를 단계를 포함한다.In addition to the above method and user equipment, the registration request is a second registration request, the method comprising: detecting failure of the first registration request to the PLMN; determining that the failure of the first registration request is the maximum number of registration failures; blocking registration with the PLMN for a predefined duration in response to the failure of the first registration request being the maximum number of registration failures; determining that the failure of the first registration request is due to a signal related state of a connection between the UE and a cell of the PLMN; and identifying, in the data store of PLMN information, the blocked PLMN due to the signal related condition.
전술한 방법 및 사용자 장비에 추가하여, 제1 등록 요청의 실패가 신호 관련 상태로 인한 것이라고 결정하는 단계는 UE와 PLMN의 셀 간의 연결의 신호 관련 특성을 측정하는 단계와; 그리고 연결의 신호 관련 특성이 임계값을 충족하지 못하는 것으로 결정하는 단계를 포함한다.In addition to the above method and user equipment, determining that the failure of the first registration request is due to a signal-related condition includes: measuring a signal-related characteristic of a connection between the UE and a cell of the PLMN; and determining that the signal-related characteristic of the connection does not satisfy the threshold.
