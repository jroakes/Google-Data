Abstract
本開示の態様は、メディアコンテンツアイテムの宛先を記述するデータを、ユーザコンピューティングデバイスによって受信することを含む、コンピュータ実装方法を対象とする。例示的な宛先は、メディアコンテンツアイテムおよびデジタルロケーション(たとえば、ウェブサイト、ソーシャルネットワーキングページなど)を含むメッセージの受信側のロケーションを含むことができる。この方法は、ユーザコンピューティングデバイスを含むコンピューティングシステムによって、メディアコンテンツアイテムの宛先を記述するデータに基づいて、1つまたは複数のメディアコンテンツアイテムを選択するステップを含むことができる。メディアコンテンツアイテムの宛先を考慮することによって、より関連性があり、および/または適切であるメディアコンテンツアイテムを選択することができる。選択されたメディアコンテンツアイテムは、ユーザコンピューティングデバイスによって動的キーボードインターフェースに表示するために提供することができる。
Aspects of the present disclosure are directed to computer implementation methods comprising receiving data describing the destination of a media content item by a user computing device. An exemplary destination can include the recipient's location of a message, including media content items and digital locations (eg, websites, social networking pages, etc.). This method can include the step of selecting one or more media content items based on the data describing the destination of the media content item by the computing system including the user computing device. By considering the destination of the media content item, you can select the media content item that is more relevant and / or appropriate. The selected media content item can be provided for display on the dynamic keyboard interface by the user computing device.

モバイルデバイスのユーザ間で通信するための従来のアプローチは、単に、SMS、ソーシャルネットワーキングアプリケーションを介したメッセージング、または「テクスティング」に依存し得る。インターネットまたはモバイルデバイスのユーザは、これらの様々な媒体を介してメッセージを交換し得る。しかしながら、時々、ユーザは、GIF(グラフィックスインターチェンジフォーマット)などのメディアコンテンツ、または静止画像やアニメーション画像のセットを含む画像ファイルを介して通信することを望む場合がある。ユーザは、インターネットでGIFを検索し、オペレーティングシステムのネイティブウェブブラウザを介してGIFをコピーし、様々なメッセージングアプリケーションにGIFを貼り付け得る。これらの従来のシステムは、リソースを費やすことなく、または手動の介入を必要とすることなく、動的インターフェース内でカテゴリ化されたコンテンツを提供することにはあまり適していない。さらに、そのような従来の検索アプローチは、メディアコンテンツの意図された宛先に関する文脈的な認識が不足している。 Traditional approaches to communicating between users of mobile devices may simply rely on SMS, messaging via social networking applications, or "texting." Users of the Internet or mobile devices may exchange messages via these various media. However, from time to time, users may wish to communicate via media content such as GIFs (graphics interchange formats), or image files containing a set of still or animated images. Users can search for GIFs on the Internet, copy GIFs through the operating system's native web browser, and paste the GIFs into various messaging applications. These traditional systems are not well suited to provide categorized content within dynamic interfaces without wasting resources or requiring manual intervention. Moreover, such traditional search approaches lack contextual awareness of the intended destination of media content.

概要
一般に、本開示は、ショートビデオなどのメディアコンテンツアイテムの改善された検索およびカテゴリ化のためのシステムおよび方法を対象とする。コンピューティングシステムは、選択されたメディアコンテンツアイテムがユーザにとってより文脈的に関連性があり、有用であるように、メディアコンテンツアイテムの宛先(たとえば、別のユーザのコンピューティングデバイス、ソーシャルネットワークプラットフォームなど)に少なくとも部分的に基づいて、メディアコンテンツアイテムを取り出し、表示することができる。
Overview In general, this disclosure covers systems and methods for improved search and categorization of media content items such as short videos. The computing system is the destination of the media content item (eg, another user's computing device, social network platform, etc.) so that the selected media content item is more contextually relevant and useful to the user. Media content items can be retrieved and displayed, at least in part.
より詳細には、本開示の態様によれば、ユーザコンピューティングデバイスは、メディアコンテンツアイテムの宛先を記述するデータを受信することができる。たとえば、ユーザは、ユーザコンピューティングデバイス上のメッセージングアプリケーションまたはソーシャルネットワークアプリケーションにアクセスすることができる。コンピューティングシステムは、メディアコンテンツアイテムの宛先に少なくとも部分的に基づいて、1つまたは複数のメディアコンテンツアイテムを選択することができる。コンピューティングシステムは、ユーザコンピューティングデバイスによる表示のために(たとえば、宛先に送信するためのユーザによる選択のために)、メディアコンテンツアイテムを提供することができる。したがって、コンピューティングシステムは、ユーザがメディアコンテンツアイテムを送信する準備をしている場所に基づいて、ユーザにとってより文脈的に関連性があり、有用であるメディアコンテンツアイテムを選択することができる。 More specifically, according to aspects of the present disclosure, the user computing device can receive data describing the destination of the media content item. For example, a user can access a messaging application or a social network application on a user computing device. The computing system can select one or more media content items based at least in part on the destination of the media content item.
