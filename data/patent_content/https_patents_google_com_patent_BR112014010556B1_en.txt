Abstract
MÉTODOS E APARELHOS PARA AUTENTICAÇÃO DE USO DE DADOS DE AMBIENTE DE ÁREA DE TRABALHO HÍBRIDA. Um método em um dispositivo eletrônico portátil (100) de autenticação de capacidades de uso de dados de rede de área ampla melhoradas. Uma comunicação de dados entre o dispositivo eletrônico portátil e uma rede de área ampla (104) é detectada, juntamente com um começo de um ambiente híbrido de sistema de operação duplo (111). O ambiente híbrido de sistema de operação duplo pode incluir um primeiro ambiente de sistema operacional (114) operável para comunicação com a rede de área ampla em uma taxa de dados restrita e um segundo ambiente de sistema operacional (116) operável para comunicação com a rede de área ampla na taxa de dados restrita ou em uma taxa de dados melhorada. Uma verificação quanto a se o dispositivo eletrônico portátil está autorizado a se comunicar com a rede de área ampla na taxa de dados melhorada então é realizada. Quando verdadeiro, o segundo ambiente de sistema operacional pode ser configurado para comunicação com a rede de área ampla na taxa de dados melhorada.METHODS AND APPLIANCES FOR AUTHENTICATION OF DATA USE OF THE HYBRID DESKTOP ENVIRONMENT. A method in a handheld electronic device (100) of authenticating enhanced wide area network data usage capabilities. A data communication between the handheld electronic device and a wide area network (104) is detected, along with a start of a dual operating system hybrid environment (111). The dual operating system hybrid environment may include a first operating system environment (114) operable for communication with the wide area network at a restricted data rate and a second operating system environment (116) operable for communication with the network. area at the restricted data rate or at an improved data rate. A check as to whether the portable electronic device is allowed to communicate with the wide area network at the improved data rate is then performed. When true, the second operating system environment can be configured to communicate with the wide area network at the improved data rate.

Além destes canais baseados em voz, as pessoas podem se comunicar através de correio eletrônico, envio de mensagem de texto, videoconferências e envio de mensagem de multimídia, da mesma forma.[002] Communication technology is constantly evolving. [016] Em um dispositivo eletrônico portátil explicativo configurado de acordo com uma ou mais modalidades da invenção, um ou mais processadores dispostos no dispositivo são configurados para operação em um ambiente híbrido de sistema operacional duplo. Um primeiro ambiente de sistema operacional é ativo durante uma operação normal. Contudo, em certos casos de uso, tal como quando o dispositivo é acoplado a um componente de hardware periférico tendo uma licença de ambiente híbrido de sistema operacional duplo, o dispositivo pode entrar em um segundo ambiente de sistema operacional tendo capacidades de uso de dados melhoradas.[016] In a portable electronic explanatory device configured in accordance with one or more embodiments of the invention, one or more processors arranged in the device are configured for operation in a dual operating system hybrid environment. A first operating system environment is active during normal operation. However, in certain use cases, such as when the device is coupled to a peripheral hardware component having a dual operating system hybrid environment license, the device may enter a second operating system environment having improved data usage capabilities. .
[017] Em uma modalidade, o ambiente híbrido de sistema operacional duplo é referido como um ambiente “WebTop”, pelo fato de o dispositivo ter acesso a dois ambientes de sistema operacional simultâneos. O primeiro ambiente de sistema operacional é um ambiente operacional móvel padrão, em que o dispositivo é configurado para interagir com a rede de área ampla usando taxas de dados de rede de área ampla e modos de uso padrões. O segundo ambiente de sistema operacional dá origem a uma taxa de dados de uso melhorada, pelo fato de o segundo ambiente de sistema operacional incluir um ambiente de área de trabalho de janela múltipla, pleno, melhorado, onde o dispositivo pode acessar um navegador da web de classe de área de trabalho e aplicativos da web, os quais são similares àqueles normalmente encontrados apenas em um computador pessoal. Neste segundo modo de operação, o dispositivo também roda o primeiro ambiente de sistema operacional, e, assim sendo, apresenta uma ou mais janelas dedicadas que exibem o conteúdo e resultados de etapas de operação no primeiro ambiente. Estas janelas podem ser referidas quanto à “Vista Móvel” do WebTop. Um usuário pode começar, parar ou interagir com os aplicativos de primeiro ambiente dentro de uma janela de Vista Móvel. O ambiente híbrido de sistema operacional duplo permite que o usuário acesse uma experiência de navegação na web de computador de mesa pleno com um dispositivo móvel, por exemplo, a visualização de versões de área de trabalho plena de websites de Internet que incluem websites baseados em Adobe Flash 10™ através da estrutura de navegador da web embutido e de aplicativo da web de dispositivo eletrônico portátil.[017] In one embodiment, the dual OS hybrid environment is referred to as a “WebTop” environment, because the device has access to two simultaneous OS environments. The first operating system environment is a standard mobile operating environment where the device is configured to interact with the wide area network using standard wide area network data rates and usage modes. The second OS environment gives rise to an improved usage data rate in that the second OS environment includes an enhanced, full, multi-window desktop environment where the device can access a web browser class desktop and web applications, which are similar to those normally found only on a personal computer. In this second mode of operation, the device also runs the first operating system environment, and therefore presents one or more dedicated windows that display the contents and results of operating steps in the first environment. These windows can be referred to as the “Mobile View” of WebTop. A user can start, stop or interact with first environment applications within a Mobile View window. The dual OS hybrid environment allows the user to access a full desktop web browsing experience with a mobile device, e.g. viewing full desktop versions of Internet websites that include Adobe-based websites Flash 10™ through embedded web browser and handheld electronic device web application framework.
[018] Pela natureza de seu projeto, os aplicativos de WebTop operando no segundo ambiente de sistema operacional transferem (via download) ordens de magnitude a mais de dados do que o fazem os aplicativos móveis operando no primeiro ambiente de sistema operacional. Assim sendo, esses aplicativos de WebTop requerem uma taxa de uso de dados melhorada. As modalidades dos métodos descritos abaixo antecipam que nem todo usuário tendo um dispositivo eletrônico portátil capaz de WebTop terá assinaturas permitindo a taxa de uso de dados melhorada a partir de uma rede de área ampla, tal como uma rede de comunicação celular ou móvel. Assim sendo, as modalidades da invenção descritas abaixo proveem métodos e sistemas para se permitirem taxas de uso de dados de WebTop e/ou restringindo um acesso a dados de rede de área ampla para apenas aqueles usuários que estão autorizados a um recurso em particular incluído em sua rede de área ampla ou conta de celular. Por exemplo, uma taxa de uso de dados melhorada pode ser permitida apenas quando o usuário tentar acessar a rede de dados de área ampla usando o segundo ambiente de sistema operacional e é autorizada de acordo com as etapas de método de autorização. Quando não autorizado, o usuário ainda pode ter permissão para acessar as taxas de uso de dados de rede de área ampla convencionais na Vista Móvel, sem interrupção.[018] By the nature of their design, WebTop applications operating in the second operating system environment transfer (via download) orders of magnitude more data than mobile applications operating in the first operating system environment do. Therefore, these WebTop applications require an improved data usage rate. The embodiments of the methods described below anticipate that not every user having a WebTop capable handheld electronic device will have subscriptions allowing for improved data usage rate over a wide area network, such as a cellular or mobile communication network. Accordingly, the embodiments of the invention described below provide methods and systems for allowing WebTop data usage fees and/or restricting wide area network data access to only those users who are authorized to a particular resource included in your wide area network or cellular account. For example, an improved data usage rate can be allowed only when the user tries to access the wide area data network using the second OS environment and is authorized according to the authorization method steps. When unauthorized, the user may still be allowed to access conventional wide area network data usage rates in Mobile View without interruption.
[019] Em uma modalidade, um método em um dispositivo eletrônico portátil de autenticação de capacidades de uso de dados de rede de área ampla melhoradas inclui o estabelecimento de comunicação de dados entre um dispositivo de comunicação operável no dispositivo eletrônico portátil e uma rede de área ampla, tal como uma rede celular. Quando uma certa condição é encontrada, tal como um dispositivo de hardware periférico tendo uma chave de licença de WebTop sendo acoplado ao dispositivo eletrônico portátil, o dispositivo pode começar o ambiente híbrido de sistema operacional duplo pela abertura de um segundo ambiente de sistema operacional, além do primeiro ambiente de sistema operacional que já está em operação. Como uma ilustração, o primeiro ambiente de sistema operacional pode ser operável para comunicação com a rede de área ampla em uma taxa de dados restrita, enquanto o segundo ambiente de sistema operacional operável pode ser configurado para comunicação com a rede de área ampla na taxa de dados restrita ou em uma taxa de dados melhorada.[019] In one embodiment, a method in a handheld electronic device of authenticating improved wide area network data usage capabilities includes establishing data communication between a communication device operable in the handheld electronic device and an area network. wide, such as a cellular network. When a certain condition is met, such as a peripheral hardware device having a WebTop license key being attached to the portable electronic device, the device can start the dual OS hybrid environment by opening a second OS environment in addition to of the first operating system environment that is already in operation. As an illustration, the first operating system environment may be operable to communicate with the wide area network at a constrained data rate, while the second operating system environment may be operable to communicate with the wide area network at a constrained data rate. restricted data or at an improved data rate.
