Hintergrundbackground
Diese Beschreibung bezieht sich darauf, Benutzern, die Anfragen an Internet-Suchmaschinen übermitteln, alternative Anfragen vorzuschlagen.These Description refers to forwarding requests to Internet search engines to users to suggest alternative inquiries.
Internetsuchmaschinen haben das Ziel Ressourcen (zum Beispiel Webseiten, Bilder, Textdokumente, Multimediakontext), die für die Bedürfnisse des Benutzers relevant sind, zu identifizieren, und Information über die Ressourcen in einer Weise, die für den Benutzer am nützlichsten ist, darzustellen. Internetsuchmaschinen geben eine Gruppe von Suchergebnissen zurück in Antwort auf eine vom Benutzer übermittelte Anfrage. Leider kann es sein, dass die Begriffe einer vom Benutzer übermittelten Anfrage nicht gut zusammenpassen mit dem, was der Benutzer im Sinn hat, zum Beispiel, wenn es eine Mehrdeutigkeit in der Bedeutung der Anfragebegriffe gibt. Selbst wenn die zurückgegebenen Suchergebnisse objektiv gesehen relevant sind für die vom Benutzer übermittelte Anfrage, können die Ergebnisse nicht relevant sein für die wirklichen subjektiven Bedürfnisse des Benutzers. Wenn ein Benutzer nicht zufrieden ist mit dem für eine Anfrage zurückgegebenen Suchergebnis, kann der Benutzer versuchen, die ursprüngliche Anfrage zu verfeinern, um diese den Bedürfnissen des Benutzers anzupassen.Internet search engines have the goal resources (for example web pages, pictures, text documents, Multimedia context), which for the needs of User are relevant to identify, and information about Resources in a way that is appropriate for the User most useful is to represent. Internet search engines give a set of search results back in response to a request submitted by the user. Unfortunately it may be that the terms are transmitted by the user Request does not match well with what the user in mind has, for example, if there is an ambiguity in the meaning the query terms are. Even if the returned search results are objective are relevant for the user submitted Request, can the results may not be relevant to the real subjective Needs of the User. If a user is not satisfied with the request returned Search result, the user can try the original Refine inquiry to suit the needs of the user.
Einige Suchmaschinen stellen einem Benutzer vorgeschlagene alternative Anfragen bereit, die die Suchmaschine als verwandt mit der ursprünglichen Anfrage des Benutzers identifiziert. Oft werden diese vorgeschlagenen alternativen Anfragen in der Nähe der Suchergebnisse der ursprünglichen Anfrage dargestellt (zum Beispiel unten auf einer Webseite mit Suchergebnissen). Jedoch kann, aufgrund begrenzter Verfügbarkeit vom Platz auf Suchergebnis-Webseiten, die Anzahl an vorgeschlagenen alternativen Anfragen, die angezeigt werden können, begrenzt sein. Des Weiteren kann die Beziehung zwischen der ursprünglichen Anfrage und den vorgeschlagenen alternativen Anfragen dem Benutzer nicht ersichtlich sein, was den Benutzer entmutigen kann, irgendeine der vorgeschlagenen alternativen Anfragen auszuwählen.Some Search engines provide a suggested alternative to a user Requests prepared by the search engine as related to the original Request of the user identified. Often these are proposed alternative requests nearby the search results of the original Request shown (for example, at the bottom of a webpage with search results). However, due to limited availability of space on search result web pages, the number of suggested alternative requests that are displayed can be be limited. Furthermore, the relationship between the original Request and the proposed alternative requests to the user not be apparent, which can discourage the user, any of the to select proposed alternative requests.
