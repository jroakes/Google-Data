Background Vehicles such as automobiles, motorcycles, aircraft and ships may include one or more computing systems to perform functions and provide information, entertainment, assistance and / or environmental control to the occupants. For example, an automobile may be an entertainment system for providing entertainment content, a navigation system for providing information and navigation assistance, a temperature control system for heating or cooling a passenger compartment, and various components or features of an automobile (eg,). , Sun roof or sunshade) may be included, or an "infotainment system" that performs some or all of these above functions. Modern vehicles are equipped with an automotive head unit (AHU) with a display device (eg, an presence-sensitive display) and an operating system and a compute engine configured to run one or more applications. Overview In general, the techniques disclosed are directed towards enabling computing devices to process queries (eg, voice or text queries) and provide query answers without the need for an active network connection. Be done. According to the disclosed technique, a computing device can detect a query through the input device of the computing device and output the requested query answer based on the detected query. The computing device does not need to send a request to the remote computing system and wait for a response from the remote computing system after receiving the query. Instead of querying the remote computing system directly, various aspects of the technique of the present disclosure allow the computing device to query using the local query answer cache (ie, the query answer cache stored locally on the computing device). It may be possible to generate an answer. The query answer cache may include the predicted queries and query answers retrieved from the query answer cache of the remote computing system, thereby detecting while the computing device has an untrusted internet connection. Allows you to respond to queries made. The query answer cache may contain predicted queries and query answers that the user is expected to ask, thereby improving traditional offline query support. Alternatively or in addition, by using such a query answer cache, the time required for the computing device to provide the query answer, even if the computing device has a trusted internet connection. Can be shortened. Thus, the disclosed techniques can reduce query response latency to reduce or eliminate the network data used when a user serves a query, thereby reducing the search efficiency of the computing device. May be improved.
