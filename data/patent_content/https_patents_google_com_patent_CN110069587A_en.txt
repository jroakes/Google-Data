Abstract
The invention discloses a kind of computer implemented method and systems for prefetching local page data from long-range mapped system to be shown in client computing devices.The user preference data acquired from the various data sources for including the application executed on the client terminal device, online or local user profile and other sources can be analyzed to generate the request for obtaining local page data from the long-range mapped system.The user preference data can refer to pictorial map characteristic, such as business, park or historical sights, the feature of the personal interest in geographical location and user with user preference.Such as, in the case where the user indicates that geographical preference is " Boston " and personal interest is " family wine brewing ", the system and method can request the local page data of all family wine brewing or the relevant character form of a map of craft beer near for Boston.
