Abstract
The system and method for providing the voice thread management in the environment based on voice activation packet.A kind of natural language processor (" NLP ") component can receive the packet including the first input audio signal and be parsed to it to identify the first request and the first triggering keyword.A kind of direct action Application Programming Interface (" API ") can generate the first action data structure of the parameter with the first action of restriction.The NLP components can receive the second input audio signal and be parsed to it to identify the second request and the second triggering keyword, and can generate the second action data structure with the parameter for limiting the second action.First and second action data can be generated as pond data structure by pond component, and can the pond data structure be transmitted to ISP's computing device so that ISP's computing device executes the operation limited by the pond data structure.
