ALLGEMEINER STAND DER TECHNIKGENERAL PRIOR ART
Verschiedene Systeme gestatten es Benutzern, Karten unterschiedlicher Standorte auf der Anzeige eines Computergerätes anzusehen. Außerdem gestatten viele dieser Systeme es Benutzern, zu manipulieren, wie die Karte angesehen wird. So stellen zum Beispiel einige Kartensysteme Tools bereit, die es einem Benutzer ermöglichen, eine Karte „zu verschieben”, wie durch Auswählen und Ziehen der Karte oder durch die Auswahl eines Punktes und die erneute Darstellung der Karte, die an dem ausgewählten Punkt auf der Anzeige zentriert ist. Einige Systeme können es einem Benutzer auch gestatten, die Karte heranzuzoomen oder zu verkleinern.Various systems allow users to view maps of different locations on the display of a computing device. In addition, many of these systems allow users to manipulate how the card is viewed. For example, some card systems provide tools that allow a user to "move" a card, such as by selecting and dragging the card, or by selecting a point and re-presenting the card at the selected point on the display is centered. Some systems may also allow a user to zoom in or out of the map.

DETAILLIERTE BESCHREIBUNGDETAILED DESCRIPTION
Überblickoverview
Die Technologie bezieht sich im Allgemeinen darauf, den Benutzern nützliche Kartendaten auf unterschiedlichen Zoomniveaus bereitzustellen. Wie vorstehend erwähnt, kann ein Benutzer eine Karte heranzoomen oder verkleinern, wodurch die dem Benutzer angezeigte geographische Region verändert wird. In einem Aspekt der Technologie kann dem Benutzer ein Satz geographischer Standorte auf der Karte dargestellt werden, der von dem derzeitigen Zoomniveau abhängt, in dem die Karte betrachtet wird.The technology generally refers to providing users with useful map data at different zoom levels. As mentioned above, a user may zoom in or out of a map, thereby changing the geographic region displayed to the user. In one aspect of the technology, the user may be presented with a set of geographic locations on the map that depends on the current zoom level at which the map is viewed.
Als Antwort auf die Anfrage des Benutzers kann das Client-Gerät des Benutzers die Karteninformationen von einem entfernten Standort, z. B. einem Server-Computer oder einem lokalen Speicher, abrufen. Das Client-Gerät kann dann eine Anzeige einer Karte übertragen. Zusätzlich zu den auf der Karte dargestellten geographischen Daten können dem Benutzer weitere kartenbezogene Informationen dargestellt werden. So kann beispielsweise der Benutzer vielleicht einen oder mehrere geographische Standorte auf der Karte auswählen, um Zensus- oder Vermessungsdaten anzusehen, die mit den ausgewählten geographischen Regionen verbunden sind. In einem anderen Beispiel können dem Benutzer Fotografien oder anderer vom Benutzer erstellter Inhalt dargestellt werden, der im Zusammenhang mit dem ausgewählten geographischen Standort steht.In response to the user's request, the user's client device may retrieve the map information from a remote location, e.g. A server computer or local store. The client device can then transmit a display of a card. In addition to the geographic data presented on the map, further map related information may be presented to the user. For example, the user may, for example, select one or more geographic locations on the map to view census or surveying data associated with the selected geographic regions. In another example, the user may be presented with photographs or other user-created content related to the selected geographic location.
In Übereinstimmung mit einem Aspekt der Technologie kann eine Hierarchie geographischer Standorte erstellt werden, auf die zugegriffen werden kann, um die besonderen geographischen Standorte festzustellen, die dem Benutzer auf unterschiedlichen Zoomniveaus dargestellt werden sollen. Auf einem niedrigen Zoomniveau können verhältnismäßig große geographische Standorte, wie Kontinente oder Länder, auf der Karte angezeigt werden. Wenn der Benutzer die Karte jedoch heranzoomt, können sich die dargestellten geographischen Standorte automatisch ändern. So können beispielsweise die dem Benutzer angezeigten geographischen Standorte von Standorten auf Landesebene zu Standorten auf Bundesland- und Provinzebene wechseln. Der Benutzer kann dann in der Lage sein, spezifische Bundesländer und Provinzen auf der Karte auszuwählen, um Informationen über die ausgewählten Bundesländer und Provinzen anzusehen. Dementsprechend kann sich die Granularität der dem Benutzer dargestellten Informationen automatisch ändern, während der Benutzer die Karte heranzoomt oder sie verkleinert.In accordance with one aspect of the technology, a hierarchy of geographic locations may be created that can be accessed to determine the particular geographic locations that are to be presented to the user at different zoom levels. At a low zoom level, relatively large geographical locations, such as continents or countries, can be displayed on the map. However, if the user zooms in on the map, the displayed geographic locations may change automatically. For example, the geographic locations displayed to the user can switch from locations at the state level to federal state and provincial locations. The user may then be able to select specific states and provinces on the map to view information about the selected states and provinces. Accordingly, the granularity of the information presented to the user may change automatically as the user zooms or shrinks the map.
In einem Beispiel kann ein Server-Computer oder ein Client-Computer festlegen, welche geographischen Standorte dem Benutzer angezeigt werden sollen, indem er die Größe der geographischen Standortkandidaten mit der Größe des Ansichtsfensters auf dem gegenwärtigen Zoomniveau vergleicht. Diese geographischen Standortkandidaten können aus einer Hierarchie geographischer Standorte ausgewählt werden, in denen geographische Stammstandorte in geographische Ablegerstandorte aufgespalten werden können.In one example, a server computer or a client computer may determine which geographic locations to display to the user by comparing the size of the geographic location candidates to the size of the viewport at the current zoom level. These geographic site candidates can be selected from a hierarchy of geographic locations where geographic root sites can be split into geographic offshoot locations.
Auf der Grundlage des Vergleichs der Größen der geographischen Standorte kann der Server oder Client-Computer einen Satz geographischer Standorte auswählen, die der Benutzer leicht ansehen kann. So können beispielsweise die geographischen Standorte so ausgewählt werden, dass die Karte mehr als nur einige Standorte auf der Karte anzeigt, aber keine große Anzahl sehr kleiner Standorte. In einem Aspekt der offenbarten Technologie kann das System bestimmte geographische Standorte innerhalb der Hierarchie bestimmten Zoomniveaus zuweisen, um dem Benutzer einen passenden Satz geographischer Standorte bereitzustellen.Based on the comparison of geographic location sizes, the server or client computer can select a set of geographic locations that the user can easily view. For example, geographic locations can be selected so that the map displays more than just a few locations on the map, but not a large number of very small locations. In one aspect of the disclosed technology, the system may assign certain geographic locations within the hierarchy to certain zoom levels to provide the user with an appropriate set of geographic locations.
In einem anderen Aspekt kann mit der Karte verbundener Inhalt, wie Fotografien mit Metadaten geographischer Standorte, dem Benutzer zusammen mit einer Liste von Benutzern angezeigt werden, die die Fotografien zur Verfügung gestellt haben. In einem Beispiel kann dem Benutzer eine Liste der besten Fotografien und der besten Fotobeitragenden angezeigt werden. Diese Liste mit den besten Fotografien und den besten Fotobeitragenden kann dem geographischen Standort entsprechen, der z. Z. vom Benutzer in einem Kartenansichtsfenster angesehen wird. Die Liste der besten Fotografien und der besten Fotobeitragenden kann dann automatisch aktualisiert werden, wenn der Benutzer sich auf der Karte bewegt oder sie zoomt. Auf diese Art können dem Benutzer Echtzeitdaten über die bestimmte geographische Region dargestellt werden, die vom Benutzer angesehen wird. Außerdem kann der Benutzer bestimmte geographische Standorte auf der Karte auswählen. Wie vorstehend erörtert, können diese geographischen Standorte basierend auf einem Vergleich des gegenwärtigen Zoomniveaus mit einer Hierarchie geographischer Standorte ausgewählt werden.In another aspect, content associated with the map, such as photographs with metadata of geographic locations, may be displayed to the user along with a list of users who provided the photographs. In one example, the user may be presented with a list of the best photographs and the best photographers. This list of the best photographs and the best photographers may correspond to the geographic location, e.g. Z. viewed by the user in a map viewport becomes. The list of best photographs and best photographers can then be updated automatically as the user moves or zooms on the map. In this way, the user can be presented with real-time data about the particular geographic region viewed by the user. In addition, the user can select specific geographic locations on the map. As discussed above, these geographic locations may be selected based on a comparison of the current zoom level with a geographic location hierarchy.

In Übereinstimmung mit einem Aspekt können die geographischen Standorte der auf den 9–11 gezeigten Fotografien in Übereinstimmung mit einem räumlichen Index gespeichert werden. Insbesondere können die die vorstehend erörterten Karten in eine Hierarchie mit Rasterfeldern unterteilt werden, in denen jeder Punkt auf der Karte einer Reihe Rasterfelder zugewiesen wird, die diesen bestimmten Punkt abdecken. In einem Beispiel kann die Erde auf eine zweidimensionale Karte, z. B. unter Verwendung einer Mercator-Projektion, projiziert werden. Eine Hierarchie von Rasterfeldern kann auf die Karte gezeichnet werden, indem die globale Karte auf Zoomniveau null in vier gleiche Quadrate und danach jedes dieser Quadrate in nochmals vier gleiche Quadrate unterteilt wird. Indem weiterhin jedes neu erstellte Quadrat in vier gleiche Quadrate unterteilt wird, bekommt man die Rasterfeldhierarchie. Dementsprechend wird jeder Punkt innerhalb des Diagramms auf der Karte durch ein Quadrat auf jeder Ebene der Hierarchie abgedeckt. Jedes Rasterfeld kann durch eine Rasterfeldkennung referenziert werden. So kann beispielsweise jede Rasterfeldkennung die Form eines 64-Bit-Datensatzes annehmen. In Übereinstimmung mit einem Aspekt können Rasterfelder, die nahe aneinander liegen, Rasterfeldkennungen erhalten, die im Wert nahe beieinander liegen. Auf diese Art können die Rasterfeldkennungen schnell nach einem bestimmten Gebiet auf der Karte durchsucht werden.In accordance with one aspect, the geographical locations of the on the 9 - 11 photographs stored in accordance with a spatial index. In particular, the maps discussed above may be subdivided into a hierarchy with grids in which each point on the map is assigned to a series of grids covering that particular point. In one example, the earth may be printed on a two-dimensional map, e.g. Using a Mercator projection. A hierarchy of grids can be drawn on the map by dividing the global map to zero zoom level into four equal squares, and then each of these squares into four equal squares again. By further dividing each newly created square into four equal squares, one obtains the grid hierarchy. Accordingly, each point within the chart on the map is covered by a square on each level of the hierarchy. Each grid can be referenced by a grid identifier. For example, each grid identifier may take the form of a 64-bit data set. In accordance with one aspect, grids that are close to one another may receive grid identifiers that are close in value to each other. In this way, the grid identifiers can be quickly searched for a specific area on the map.
