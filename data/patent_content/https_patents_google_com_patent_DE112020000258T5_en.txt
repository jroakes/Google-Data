HINTERGRUNDBACKGROUND
Heutzutage ist es üblich, dass Menschen online Waren einkaufen, anstatt in Ladengeschäfte zu gehen. Während Personen verschiedene Webseiten und Domains besuchen, um einzukaufen, werden Informationen über die Personen häufig von Dritten mithilfe von Cookies erfasst. Cookies ermöglichen es Dritten zu verfolgen, welche Webseiten eine Person besucht, wie oft die Person jede Webseite besucht, wie lange die Person auf jeder Webseite verweilt, bevor sie eine neue Webseite ansieht, alle Auswahlen, die die Person auf jeder Webseite trifft, den Inhalt jeder Webseite, alle Dateneingaben, die die Person auf jeder Webseite vornimmt, usw. Die Drittanbieter sind häufig Inhaltsanbieter, die die von Cookies gesammelten Daten verwenden, um benutzerspezifische Inhalte für Benutzer, basierend auf den durch die Cookies für jeden Benutzer gesammelten Daten auszuwählen. Der benutzerbezogene Inhalt kann einem Benutzer erscheinen, wenn der Benutzer eine Webseite ansieht, die möglicherweise nicht mit dem Inhalt, der dem Benutzer angezeigt wird, in Zusammenhang steht.Nowadays it is common for people to shop for goods online instead of going to physical stores. While people visit different websites and domains to shop, information about the people is often collected by third parties using cookies. Cookies allow third parties to keep track of which web pages a person visits, how often the person visits each web page, how long the person dives on each web page before viewing a new web page, all the choices that the person makes on each web page, the content each website, all the data that the person enters on each website, etc. The third-party providers are often content providers that use the data collected by cookies to select user-specific content for users based on the data collected by the cookies for each user. The user-related content may appear to a user when the user views a web page that may not be related to the content that is displayed to the user.
Um unerwünschtes Tracking und folglich eine benutzerorientierte Inhaltsauswahl durch Drittanbieter von Inhalten zu stoppen, bieten Websites und Webbrowser Benutzern häufig eine Option, die Benutzer fragt, ob der Benutzer Cookies aktivieren oder deaktivieren möchte. Selbst wenn ein Benutzer eine Option zum Deaktivieren von Cookies auswählt, können Drittanbieter von Inhalten die Auswahl des Benutzers ignorieren und trotzdem Cookies verwenden oder die Auswahl des Benutzers zum Deaktivieren von Cookies umgehen, indem sie Techniken zur Neuidentifizierung verwenden (z.B. deterministische oder probabilistische Ansätze, E-Mail-basierte Identitätssynchronisierung, Telefon-basierte Identitätssynchronisierung, Server-zu-Server-Synchronisierung, Remote-Prozeduraufrufe usw.). Diese Neuidentifizierungstechniken ermöglichen es Drittanbietern von Inhalten, die Aktivitäten von Benutzern zu verfolgen, nachdem die Benutzer die Option zum Deaktivieren von Cookies ausgewählt haben und folglich denken, dass sie nicht verfolgt werden oder benutzerorientierte Inhalte erhalten.To stop unwanted tracking and, consequently, user-oriented content selection by third party content providers, websites and web browsers often provide users with an option that asks users whether the user wants to enable or disable cookies. Even if a user selects an option to disable cookies, third party content providers may ignore the user's choice and still use cookies, or bypass the user's choice to disable cookies by using re-identification techniques (e.g., deterministic or probabilistic approaches, E. -Mail-based identity synchronization, phone-based identity synchronization, server-to-server synchronization, remote procedure calls, etc.). These re-identification techniques allow third-party content providers to track users' activities after users have selected the option to disable cookies and consequently think they will not be tracked or receive user-oriented content.
Es kann für ein Computersystem oder einen Webbrowser schwierig sein, festzustellen, ob Benutzer von Inhaltsanbietern von Drittanbietern verfolgt werden, nachdem die Benutzer eine Option zum Deaktivieren von Cookies ausgewählt haben. Derselbe Inhalt kann von Computergeräten angefordert und Benutzern präsentiert werden, unabhängig davon, ob die Benutzer verfolgt werden. Ein Benutzer kann beispielsweise einen Browser öffnen und eine Option zum Deaktivieren von Cookies auswählen. Der Benutzer kann eine Webseite besuchen, auf der Autos diskutiert werden, und dann eine andere Webseite besuchen, auf der Inhalte, die ein Auto beschreiben, in einem Seitenband der Webseite erscheinen. Derselbe Inhalt kann als Ergebnis der Verfolgung des Benutzers von der Website über das Auto präsentiert werden oder kann zufällig ausgewählt und präsentiert werden. Frühere Systeme und Verfahren haben keine Möglichkeit zu bestimmen, wie der Inhalt ausgewählt wurde (z.B. gezielt oder zufällig ausgewählt), um festzustellen, ob der Inhaltsanbieter Benutzer verfolgt, obwohl der Benutzer eine Option zum Deaktivieren von Cookies ausgewählt hat. Während Webbrowser derzeit Drittanbieter von Inhalten blockieren können, die Benutzer verfolgen, auch wenn eine Option zum Nicht-Tracking ausgewählt wurde, können frühere Systeme und Methoden Inhaltsanbieter, die dies tun, nicht identifizieren.It can be difficult for a computer system or web browser to determine if third-party content providers are tracking users after users select an option to disable cookies. The same content can be requested by computing devices and presented to users regardless of whether the users are being tracked. For example, a user can open a browser and select an option to disable cookies. The user can visit a web page where cars are discussed and then visit another web page where content describing a car appears in a sideband of the web page. The same content can be presented as a result of tracking the user from the website through the car, or can be selected and presented at random. Prior systems and methods did not have a way of determining how the content was selected (e.g., selectively or randomly) to determine if the content provider is tracking users even though the user has selected an option to disable cookies. While web browsers can currently block third-party content providers that are tracking users even if a non-tracking option is selected, previous systems and methods cannot identify content providers that do so.
ZUSAMMENFASSUNGSUMMARY
Die hierin erörterten Systeme und Verfahren stellen ein Modell für maschinelles Lernen bereit (z.B. ein neuronales Netzwerk, eine Support-Vektor-Maschine, Random Forest usw.), das automatisch eine Wahrscheinlichkeit bestimmen kann, dass ein Inhaltsanbieter Benutzer verfolgt, obwohl die Benutzer eine Option auswählen, die andernfalls verhindern würde, dass sie verfolgt werden. Die Eingaben in das neuronale Netz können gemeinsame Merkmale sein, die ein Inhaltselement, das einem Benutzer auf einer Webseite gezeigt wird, mit anderen Webseiten, Domänen und Schlüsselwörtern hat, die Benutzer zuvor angesehen oder in einen Browser eingegeben haben. Andere Eingaben können allgemeine Webseiten, Domänen und Schlüsselwörter zwischen Benutzern sein, die auf verschiedenen Computern eine Eingabe gemacht haben, bevor sie denselben Inhalt erhalten. Jede dieser Eingaben kann mit unterschiedlichen Gewichtungen im neuronalen Netz verknüpft werden, um eine binäre Klassifikation zu bestimmen, die eine Wahrscheinlichkeit darstellt, dass ein bestimmter Inhaltsanbieter Online-Aktivitäten von Webbenutzern verfolgt. Die Gewichtungen können fein abgestimmt werden, um die Leistung des neuronalen Netzwerks zu verbessern, um eine Wahrscheinlichkeit, dass Inhaltsanbieter Benutzer verfolgen, genau zu bestimmen, wenn das neuronale Netzwerk mehr Eingaben empfängt und Ausgaben erzeugt. Ein System kann die Wahrscheinlichkeit erhalten, dass ein Inhaltsanbieter Benutzer verfolgt, die durch das neuronale Netz bestimmt werden, und bestimmen, ob die Wahrscheinlichkeit über einem vom Administrator ausgewählten Schwellenwert liegt. Wenn die Wahrscheinlichkeit über dem Schwellenwert liegt, kann das System verhindern, dass der Inhaltsanbieter den Benutzern Inhalte bereitstellt. Alle Inhaltsanforderungen von Computergeräten, die an die Inhaltsanbieter gesendet werden, von denen festgestellt wurde, dass sie Benutzer verfolgen, können an andere Inhaltsanbieter umgeleitet werden.The systems and methods discussed herein provide a machine learning model (e.g., neural network, support vector machine, random forest, etc.) that can automatically determine a likelihood that a content provider will track users even though the users have an option that would otherwise prevent them from being tracked. The inputs to the neural network may be common characteristics that an item of content presented to a user on a web page has with other web pages, domains, and keywords that users have previously viewed or entered into a browser. Other inputs can be common web pages, domains, and keywords between users who typed on different computers before receiving the same content. Each of these inputs can be linked to different weights in the neural network in order to determine a binary classification that represents a probability that a particular content provider is tracking online activities of web users. The weights can be fine-tuned to improve the performance of the neural network to accurately determine a likelihood that content providers will track users as the neural network receives more inputs and generates more outputs. A system can obtain the likelihood that a content provider is tracking users, which is determined by the neural network, and determine whether the likelihood is over is within an administrator-selected threshold. If the probability is above the threshold, the system can prevent the content provider from serving content to users. All content requests from computing devices sent to the content providers that have been determined to be tracking users can be redirected to other content providers.
Vorteilhafterweise kann ein System durch Implementieren der hierin erörterten Systeme und Verfahren automatisch Drittanbieter von Inhalten identifizieren, die Webbrowser-Benutzer verfolgen, nachdem die Benutzer eine Option ausgewählt haben, um die Anbieter daran zu hindern. Systeme, die die hierin erörterten Systeme und Verfahren nicht verwenden, können sich auf Proxys, VPNs, Benutzeragenten-Scrubbing, Fingerabdruck-Identifikationen (z.B. eindeutige Identifikationen in URL-Parametern), E-Mail-Abgleich und -Reduzierung, Telefonnummern-Abgleich und -Reduzierung usw. verlassen, um Inhaltsanbieter zu identifizieren und daran zu hindern, Benutzer zu verfolgen. Jede dieser Techniken kann verwendet werden, je nachdem, wie Dritte die Benutzer verfolgen, aber sie sind nicht gegen alle Tracking-Techniken wirksam. Außerdem ist es nicht immer möglich, diese Identifizierungs- und Präventionstechniken zu implementieren, da sie schwierig zu skalieren, zu teuer oder schwierig zu implementieren sein können.Advantageously, by implementing the systems and methods discussed herein, a system can automatically identify third party content providers that web browser users are tracking after the users select an option to prevent the providers from doing so. Systems that do not use the systems and procedures discussed herein can rely on proxies, VPNs, user agent scrubbing, fingerprint identifications (e.g. unique identifications in URL parameters), email matching and reduction, phone number matching and - Leave reduction etc to identify content providers and prevent them from tracking users. Any of these techniques can be used depending on how third parties are tracking users, but they are not effective against all tracking techniques. In addition, it is not always possible to implement these identification and prevention techniques as they can be difficult to scale, too expensive, or difficult to implement.
Glücklicherweise können die hierin beschriebenen Systeme und Verfahren verwendet werden, um genau und automatisch zu bestimmen, ob Dritte Benutzer verfolgen, unabhängig davon, wie die Dritten die Verfolgung durchführen, und dann zu verhindern, dass die Dritten weitere Inhalte an Webbrowser bereitstellen. Ein System kann dies auf der Grundlage von Eigenschaften von Inhalten, die von Inhaltsanbietern bereitgestellt werden, und Eigenschaften von Inhalten, die zuvor von Benutzern angesehen wurden, tun. Unter Verwendung eines neuronalen Netzes mit einer binären Klassifikationsausgabe kann das System zwischen zufällig ausgewählten und den Benutzern präsentierten Inhalten und Inhalten, die ein Produkt gezielter Verfolgung sind, unterscheiden, selbst wenn einem Benutzer Inhalte angezeigt werden, nachdem der Benutzer eine Website besucht hat, die ähnliche Merkmale wie die angezeigten Inhalte aufweist. Folglich müssen Drittanbieter von Inhalten nicht in die Privatsphäre von Personen eindringen, indem sie ihre Webbrowser-Aktivitäten verfolgen, da sie daran gehindert werden, Webbenutzern Inhalte bereitzustellen, nachdem sie für die Verfolgung von Benutzern identifiziert wurden, die ihre Webbrowser-Aktivitäten verborgen halten möchten.Fortunately, the systems and methods described herein can be used to accurately and automatically determine whether a third party is tracking users, regardless of how the third party is tracking, and then preventing the third party from serving further content to web browsers. A system can do this based on properties of content provided by content providers and properties of content previously viewed by users. Using a neural network with a binary classification output, the system can distinguish between content that is randomly selected and presented to users and content that is a product of targeted tracking, even if content is displayed to a user after the user has visited a website that is similar Has characteristics such as the content displayed. As a result, third party content providers do not need to invade people's privacy by tracking their web browser activities as they will be prevented from serving content to web users after they have been identified for tracking users who wish to keep their web browser activities hidden.

Ein Computerprogramm (auch bekannt als Programm, Software, Softwareanwendung, Skript oder Code) kann in jeder Form von Programmiersprache geschrieben werden, einschließlich kompilierter oder interpretierter Sprachen, deklarativer oder prozeduraler Sprachen, und es kann in jede Form, einschließlich als eigenständiges Programm oder als Modul, Komponente, Unterprogramm, Objekt oder andere Einheit, die zur Verwendung in einer Computerumgebung geeignet ist, eingesetzt werden. Ein Computerprogramm kann, muss aber nicht, einer Datei in einem Dateisystem entsprechen. Ein Programm kann in einem Teil einer Datei, die andere Programme oder Daten enthält, gespeichert werden (z.B. ein oder mehrere Skripte, die in einem Auszeichnungssprachendokument gespeichert sind), in einer einzelnen Datei, die dem fraglichen Programm zugehörig ist, oder in mehreren koordinierten Dateien (z.B., Dateien, die ein oder mehrere Module, Unterprogramme oder Codeteile speichern). Ein Computerprogramm kann eingesetzt werden, um auf einem Computer oder auf mehreren Computern ausgeführt zu werden, die sich an einem Standort befinden oder über mehrere Standorte verteilt und durch ein Kommunikationsnetzwerk miteinander verbunden sind.A computer program (also known as a program, software, software application, script, or code) can be written in any form of programming language, including compiled or interpreted languages, declarative or procedural languages, and it can be written in any form, including as a stand-alone program or as a module , Component, subroutine, object, or other entity suitable for use in a computing environment. A computer program can, but does not have to, correspond to a file in a file system. A program can be stored in part of a file that contains other programs or data (e.g., one or more scripts stored in a markup language document), in a single file associated with the program in question, or in several coordinated files (e.g., files that store one or more modules, subprograms or pieces of code). A computer program can be used to run on one computer or on several computers that are located at one location or that are distributed over several locations and are connected to one another by a communication network.
